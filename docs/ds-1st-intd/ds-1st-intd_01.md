# 前言

> 原文：[`datasciencebook.ca/foreword.html`](https://datasciencebook.ca/foreword.html)

*Roger D. Peng*

*约翰霍普金斯大学公共卫生学院*

*2022-01-04*

近年来，数据科学领域已经显著扩展和增长，吸引了来自不同方向的兴奋和兴趣。随着该领域本身的发展，对入门教育材料的需求也在增长，导致教科书、课程、博客文章和教程的激增。这本书是对这一快速增长的文献的重要贡献，但鉴于材料的广泛可用性，读者可能会倾向于问：“这本书的独特贡献是什么？”为了回答这个问题，暂时回顾一下过去几年数据科学领域的发展是有用的。

当思考数据科学时，考虑两个问题是重要的：“什么是数据科学？”以及“一个人应该如何进行数据科学？”前一个问题在广泛的学者和实践者社区中正在积极讨论，目前似乎还没有达成太多共识。然而，似乎有一个普遍的理解，即数据科学专注于用数据回答问题的更“活跃”的元素——数据处理、清洗和分析。这些元素通常高度特定于问题，可能看起来难以在不同应用中推广。尽管如此，随着时间的推移，我们已经看到一些核心元素出现，这些元素似乎在不同问题中作为有用的概念重复出现。鉴于对数据科学定义的缺乏明确共识，迫切需要像这样一本书来提出对这一领域是什么以及该领域成员参与的活动的影响的愿景。

本书首先讨论的重要概念是 tidy 数据，这是一种表格数据的格式，由 Hadley Wickham 在 2014 年的一篇论文中正式介绍给统计社区。tidy 数据组织策略已被证明是进行数据分析的有力抽象概念，这在很大程度上得益于 Tidyverse 集合中 R 包的庞大工具链。第二个关键概念是开发可重复和可审计的数据分析工作流程。由于数据可用性和我们实施复杂数据分析程序容易，现代数据分析的复杂性不断增加。此外，这些数据分析往往是决策过程的一部分，可能对个人和社区产生重大影响。因此，构建可重复的分析，其他人可以以可靠的方式研究和重复，变得至关重要。统计方法显然是数据科学中构建预测和分类模型以及做出关于未观察人群推断的重要元素。最后，因为一个领域只有在其培养一个活跃和协作的社区时才能成功，因此很明显，熟练掌握协作工具是数据科学的核心要素。

这本书将这些核心概念集中在如何以严谨的方式将它们应用于*做*数据科学。从这本书学习的学生将精通从数据中产生可靠证据的技术和原则。本书以在 tidy 数据框架内使用 R 编程语言为中心，因此采用了数据分析编码的最新进展。使用 Jupyter 笔记本进行练习立即将学生置于一个鼓励分析可审计性和可重复性的环境中。将 git 和 GitHub 整合到课程中是教授协作和社区的关键工具，这些是数据科学中至关重要的概念。

对数据科学培训的需求持续增加。大量数据可用于回答各种问题，可用的计算能力比以往任何时候都多，以及公众对数据在决策中重要性的认识，这些都促使对高质量数据科学工作的需求。本书为数据科学领域提供了一个复杂的第一介绍，并提供了实用技能和普遍原则的平衡组合。随着我们继续向学生介绍数据科学并训练他们面对不断扩大的数据科学问题，这里提出的思想将为他们提供良好的服务。
