# 关于作者

> 原文：[`datasciencebook.ca/about-the-authors.html`](https://datasciencebook.ca/about-the-authors.html)

**[蒂芙尼·蒂姆伯斯](https://tiffanytimbers.com/)** 是不列颠哥伦比亚大学统计学系的教学副教授，同时也是数据科学硕士项目（温哥华选项）的联合主任。在这些角色中，她教授并开发围绕负责任地应用数据科学解决现实世界问题的课程。她教授的最喜欢的课程之一是关于协作软件开发的研究生课程，该课程侧重于教授如何使用现代工具和工作流程创建 R 和 Python 包。

**[特雷弗·坎贝尔](https://trevorcampbell.me)** 是不列颠哥伦比亚大学统计学系的副教授。他的研究重点在于自动化的、可扩展的贝叶斯推理算法、贝叶斯非参数方法、流数据以及贝叶斯理论。他之前是麻省理工学院计算机科学与人工智能实验室（CSAIL）和信息系统与数据研究所（IDSS）的博士后研究员，由塔玛拉·布罗德里克指导，是麻省理工学院信息与决策系统实验室（LIDS）的博士候选人，在此之前，他在多伦多大学的工程科学项目中学习。

**[梅丽莎·李](https://www.stat.ubc.ca/users/melissa-lee)** 是不列颠哥伦比亚大学统计学系的教学助理教授。她教授并开发本科生统计学和数据科学课程的课程。她的工作重点在于以学生为中心的教学方法、开发与评估开放教育资源，以及促进公平、多元化和包容性倡议。
