# 5\. 谱图论

> 原文：[`mmids-textbook.github.io/chap05_specgraph/00_intro/roch-mmids-specgraph-intro.html`](https://mmids-textbook.github.io/chap05_specgraph/00_intro/roch-mmids-specgraph-intro.html)

在本章中，我们研究网络数据。我们首先定义基本的图概念。然后，我们介绍谱图理论中的思想，特别是拉普拉斯矩阵的特征值及其变分特征。我们讨论并实现了某些应用，包括社区检测。以下是本章主要部分的更详细概述。

*“背景：图论的基本概念”* 这一节介绍了图论的基本概念，涵盖了无向图和有向图。它介绍了关键定义，如顶点、边、路径、连通性以及各种特殊类型的图，如完全图、树和有向无环图。这一节还讨论了不同的矩阵表示。它讨论了拉普拉斯矩阵，强调了它在理解图连通性中的作用。最后，这一节强调了这些概念的实际应用，即使用 NetworkX Python 包进行图操作和可视化。

*“特征值的变分特征”* 这一节探讨了特征值的变分特征，提供了理解这些重要数学对象的不同视角。它首先展示了一个谱定理的证明，展示了如何使用正交变换来对角化对称矩阵。这个证明是后续探索变分特征的基础，这些特征涉及将特征值表示为优化问题的解。这一节详细介绍了特定情况，展示了如何通过涉及雷利商的优化程序来确定最大和最小的特征值。最后，它提出了 Courant-Fischer 定理。

*“拉普拉斯矩阵的谱性质”* 这一节讨论了图拉普拉斯矩阵的谱性质。它首先观察到拉普拉斯矩阵是对称的且正半定的，并且常量单位向量始终是特征值为 0 的特征向量。然后，这一节探讨了拉普拉斯矩阵与图连通性之间的关系，证明如果一个图是连通的，当且仅当第二个最小的特征值（即代数连通性）是严格正的。最后，这一节提出了第二个最小特征值的变分特征，该特征在满足中心化和归一化约束的条件下最小化分配给相邻顶点的值的平方差的和。这种特征用于解释为什么拉普拉斯矩阵的特征向量对于图绘制和揭示图的潜在几何结构是有用的。

*“应用：通过谱聚类的图划分”* 本节探讨了图划分，即将图划分为两个节点集，这两个节点集之间的连接尽可能少。它介绍了一个衡量切割质量的标准，平衡了最小化集合间边的需要与平衡集合大小需求之间的关系，以及与图等周数（或 Cheeger 常数）相关的概念。本节介绍了 Cheeger 不等式，将等周数与第二小的拉普拉斯特征值联系起来，展示了谱图理论与图划分之间的关系。本节概述了一个利用拉普拉斯特征向量的图切割算法，提供了一种启发式方法来找到具有可证明保证的良好切割，并讨论了寻找最优切割的计算挑战。最后，本节回顾了社区检测，通过应用谱性质来识别图内的子社区。

*“Erdős-Rényi 随机图和随机块模型”* 本节讨论随机图模型，重点关注非齐次 Erdős-Rényi（ER）随机图和随机块模型（SBM）。非齐次 ER 随机图是一个用于生成随机图的通用模型，其中每条边以指定的概率独立地被包含。一个特殊情况是 ER 随机图，其中不同顶点之间的所有边具有相同的出现概率。SBM 通过将顶点划分为块并分配块内和块间连接的不同概率来扩展这一概念，从而特别捕捉到同一组内节点之间连接频率更高的趋势。
