1.  附录

1.  D 数据集

学生们常常会遇到的一个问题是选择数据集。一般来说，最好避开 Kaggle、UCI 机器学习仓库和其他常用选项的数据集。从数据科学的角度来看，使用从这些来源直接获取的数据集意味着几乎所有的重要决策都已经做出，并且可能没有文档记录。从职业发展的角度来看，这并不能使你的作品集脱颖而出，因为其他人只是使用这些数据集。一些替代方案包括：

+   [AidData](https://www.aiddata.org/datasets) 提供了大量与发展研究和外国援助相关的研究数据集。

+   [Alex Cookson 的数据集](https://github.com/tacookson/data)。

+   Andrews 和 Herzberg (2012) 提供了多种数据集，这些数据集可在[这里](https://www.york.ac.uk/depts/maths/histstat/pml1/r/andrews.htm)找到。

+   [*社会科学家 API*](https://bookdown.org/paul/apis_for_social_scientists/) 提供了多种 API，可用于收集数据。

+   Bombieri 等人 (2023) 提供了一个关于 5000 多起大型食肉动物攻击人类事件的数据集。

+   大英图书馆的世界报纸[目录](https://bl.iro.bl.uk/concern/datasets/943bd083-6355-44a1-97eb-b8ff898f87d5?locale=en)包含了关于出版起始和结束年份、出版地点、变体标题和版本以及出版语言的信息。

+   BuzzFeed News 提供了其文章基础上的许多数据集的[访问权限](https://github.com/BuzzFeedNews/nics-firearm-background-checks)。

+   [加拿大市政选举数据库](https://dataverse.scholarsportal.info/dataset.xhtml?persistentId=doi:10.5683/SP2/4MZJPQ) 包含了加拿大各地市政选举的完整结果 (Lucas et al. 2020)。

+   [Congressindata](https://www.congressindata.com) 提供了关于 2005 年至 2015 年美国国会成员的数据集。

+   [Congress.gov API](https://blogs.loc.gov/law/2022/09/introducing-the-congress-gov-api/) 是关于美国国会数据，特别是法案和其他文本数据的特别有用的数据来源。

+   [COVerAGE-DB](https://osf.io/mpwjq/) 是一个全球性的 COVID-19 病例和死亡人口统计数据数据库 (Riffe et al. 2021)。

+   `cricketdata` (Hyndman et al. 2022) 提供了下载关于国际和其他主要板球比赛数据的函数。

+   [数据与故事库](https://dasl.datadescription.com) 提供了数百个数据集的访问权限。

+   [数据是复数](https://www.data-is-plural.com) 提供了一个包含有趣数据集的每周通讯，其存档可追溯到 2015 年。

+   [数据解放项目](https://www.data-liberation-project.org)专注于使用信息自由请求构建美国政府数据集。

+   [人口与健康调查](https://dhsprogram.com)（DHS）项目从 1984 年开始为 90 个国家提供调查数据。

+   Duolingo [提供](https://research.duolingo.com)其研究论文支撑的数据集访问权限。

+   《经济学人》提供了其文章支撑的许多数据集的[访问权限](https://github.com/orgs/TheEconomist/repositories)。

+   [EH.net](https://eh.net/databases/) 提供了各种有趣的历史经济数据集。

+   环保署（EPA）提供了来自[未监管污染物监测规则](https://www.epa.gov/dwucmr/occurrence-data-unregulated-contaminant-monitoring-rule#5)的发生数据。

+   [欧洲 NUTS 级别选举数据库（EU-NED）](https://eu-ned.com) 提供了 1990 年至 2020 年的国家和欧洲议会选举结果。

+   联邦储备经济数据（FRED）[提供](https://fred.stlouisfed.org)美国经济数据，并且有一个名为 `fredr` 的 R 包（Boysel 和 Vaughan 2021）用于访问 API。

+   五三八（FiveThirtyEight）提供了其文章支撑的许多数据集的[访问权限](https://github.com/fivethirtyeight/data)。

+   [Goodreads 数据集](https://sites.google.com/eng.ucsd.edu/ucsdbookgraph/home) 是 2017 年从超过两百万本书的公共数据中抓取的，包括元数据和评论（Wan 和 McAuley 2018；Wan 等人 2019）。

+   [历史社会冲突数据库](https://www.unicaen.fr/hiscod/accueil.html)提供了关于超过 20,000 次冲突的数据，主要关注欧洲（Chambru 和 Maneuvrier-Hervieu 2022）。

+   [历史统计数据](https://www.historicalstatistics.org/) 提供了历史统计数据的链接。

+   [人类死亡率数据库](https://www.mortality.org) 为多个国家提供了详细的死亡率和人口数据。

+   ICANN 的[集中式区域数据服务](https://czds.icann.org/home)在经过几天申请和审批流程后，提供对所有域名的访问权限。

+   [IPCC 数据分发中心](https://www.ipcc-data.org)。

+   爱尔兰社会科学数据档案馆拥有各种[可用](https://www.ucd.ie/issda/data/)的数据集。

+   J-PAL（阿卜杜勒·拉蒂夫·贾梅尔贫困行动实验室）维护了一个[数据集目录](https://www.povertyactionlab.org/catalog-administrative-data-sets)。

+   [NFL Savant](http://nflsavant.com/index.php) 提供了关于 NFL 的团队特定数据，包括自 2013 年以来的比赛数据、自 1999 年以来的综合数据和天气数据。

+   标记的[展示你的工作](https://themarkup.org/series/show-your-work)系列通常包括指向 GitHub 仓库的链接，这些仓库包含支撑文章的数据。其中一些值得注意的包括：[隐藏在抵押贷款审批算法中的秘密偏见](https://github.com/the-markup/investigation-redlining)。

+   马萨诸塞州水资源管理局在其[这里](https://www.mwra.com/biobot/biobotdata.htm)提供其废水 COVID-19 追踪数据，原始数据以 PDF 格式提供，可进行解析。

+   现代艺术博物馆（MoMA）使其[收藏](https://github.com/MuseumofModernArt/collection)和[展览](https://github.com/MuseumofModernArt/exhibitions)的数据集可用。

+   NASA 的[行星数据系统](https://pds.nasa.gov)。

+   [ProPublica 数据存储](https://www.propublica.org/datastore/)提供了大量关于美国的数據集，其中一些相当庞大。例如，[开放支付数据（2016）](https://www.propublica.org/datastore/dataset/cms-open-payments-data-2016)有 6GB。

+   Laouenan 等人[的著名人物数据集](https://medialab.github.io/bhht-datascape/) (2022)提供了一个从公元前 3500 年到公元 2018 年的著名人物的交叉验证数据库。

+   OECD [提供](https://data.oecd.org)经济数据。

+   ParlEE 数据集包含欧盟立法机构数百万次演讲的标注全文 (Sylvester et al. 2023)。

+   监狱政策倡议提供了许多关于美国监狱和拘留所的[数据集](https://www.prisonpolicy.org/data/)。

+   The Pudding 将支撑其文章的许多数据集[提供](https://github.com/the-pudding/data)。其中一些值得注意的包括：[裸露的真相](https://github.com/the-pudding/data/tree/master/foundation-names)，以及[美国人口普查的演变](https://github.com/the-pudding/data/tree/master/census-history)。

+   [Pushshift Reddit 数据集](https://files.pushshift.io/reddit/)是自 2015 年以来 Reddit 帖子的集合 (Baumgartner et al. 2020)。

+   [难民法律实验室](https://refugeelab.ca)以 JSON 格式提供加拿大最高法院判决的全文 (Rehaag 2023)。

+   Rijksmuseum [提供](https://data.rijksmuseum.nl)了关于其收藏的各种数据。

+   Socioeconomic High-resolution Rural-Urban Geographic Platform [(SHRUG)](http://devdatalab.org/shrug)是一个开放数据平台，提供关于印度 60 万个村庄和城镇社会经济发展的数据 (Asher et al. 2021)。

+   Tom Cardoso 的[监狱背后的偏见](https://www.theglobeandmail.com/canada/article-investigation-racial-bias-in-canadian-prison-risk-assessments/)提供了关于加拿大黑人原住民囚犯的数据。

+   跟踪（不）正义是一个追踪加拿大警察参与死亡事件的数据库（数据和正义犯罪学实验室，犯罪学和刑事司法学院，卡尔顿大学；黑帮受害者幸存者研究与创新能力中心（CRIB），多伦多大学 Factor-Inwentash 社会工作学院；加拿大公民自由协会；皇后大学伦理与技术实验室 2022）。

+   美国疾病控制与预防中心（CDC）的国家生命统计系统提供了各种数据集，包括[关联出生和婴儿死亡数据](https://www.cdc.gov/nchs/nvss/linked-birth.htm#Two_Formats)。

+   美国量刑委员会由[Kevin Wilson](https://kevinhayeswilson.com/data.html)清洗和准备的[个人罪犯数据集](https://www.ussc.gov/research/datafiles/commission-datafiles)。

+   [武装叛乱中的女性活动](https://www.waarproject.com)提供了 1946-2015 年间女性在叛乱组织中的参与度数据(Loken and Matfess 2023)。

+   华盛顿邮报提供了许多支撑其文章的[数据集](https://github.com/washingtonpost)的访问权限。特别值得注意的是[国会议员奴隶主](https://github.com/washingtonpost/data-congress-slaveowners)、[致命武力射击](https://github.com/washingtonpost/data-police-shootings)、[校园枪击事件](https://github.com/washingtonpost/data-school-shootings)以及[为什么联邦紧急管理局拒绝向美国南部深色人种灾难幸存者提供援助](https://github.com/wpinvestigative/fema_ihp_denials)。

+   [Wordbank 数据库](http://wordbank.stanford.edu)是一个公开的儿童词汇增长数据库。此外，还可以通过`wordbankr`(Braginsky 2020)访问，[Alison Presmanes Hill](https://www.apreshill.com)提供了有用的[背景信息](https://apreshill.github.io/data-vis-labs-2018/03-colors.html)和[清洗代码](https://apreshill.github.io/data-vis-labs-2018/03a-meow-cleaning.html)。

+   世界银行为[全球发展数据](https://data.worldbank.org)和[微观数据库](https://microdata.worldbank.org/index.php/home/)提供了广泛的数据。

+   耶鲁国际金融中心的数据集：[历史金融研究数据](https://som.yale.edu/centers/international-center-for-finance/data/historical-financial-research-data)和[股票市场信心指数](https://som.yale.edu/centers/international-center-for-finance/data/stock-market-confidence-indices)。
