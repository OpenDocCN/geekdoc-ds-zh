# 17.3 带有权重边的图

> 原文：[`dcic-world.org/2025-08-27/weighted-graphs.html`](https://dcic-world.org/2025-08-27/weighted-graphs.html)

| |
| --- |

考虑一个交通图：我们通常不仅对能否从一个地方到达另一个地方感兴趣，还对其“成本”感兴趣（我们可能有多种不同的成本度量：金钱、距离、时间、二氧化碳单位等）。在互联网上，我们可能关心链接的☛延迟或☛带宽。即使在社交网络中，我们也可能喜欢描述朋友之间的亲近程度。简而言之，在许多图中，我们不仅对边的方向感兴趣，还对某些抽象的数值度量感兴趣，我们称之为其权重。

在本研究的其余部分，我们将假设我们的图边具有权重。这并不使之前我们研究的内容失效：如果一个节点在无权图中是可达的，那么在有权图中它仍然是可达的。但下面我们要研究的操作只在有权图中才有意义。然而，我们可以通过给每条边赋予相同的、恒定的、正权重（比如说一个）来始终将无权图视为有权图。
