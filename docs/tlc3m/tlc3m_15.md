# 附录 B. 术语表

> 原文：[`little-book-of.github.io/maths/books/en-US/chronicles-b.html`](https://little-book-of.github.io/maths/books/en-US/chronicles-b.html)

### B1. 数与量

*数学的诞生始于区分一个和许多的行为——计数、测量、比较。这个集群汇集了使数量可见和可操作的基础概念，将手势转化为算术，将思想转化为代数。*

| 术语 | 定义 | 上下文 | 现代用法 |
| --- | --- | --- | --- |
| 数 | 代表数量、顺序或测量的抽象概念；数学的基础。 | 从早期农业社会对有形物体的计数中产生。 | 是所有数学系统的基础；整数、有理数、实数、复数等。 |
| 自然数 | 计数数集（1, 2, 3, …），用于计数对象。 | 根植于原始计数；最早的总账和鹅卵石。 | 用于离散数学、算法和组合数学。 |
| 整数 | 包括负数、零和正数（…，-2，-1，0，1，2，…）的整数。 | 为了表示债务、对立和方向而发明。 | 在编程、数论和代数中无处不在。 |
| 有理数 | 可以表示为两个整数比（\(\frac{p}{q}\)）的数。 | 由希腊几何学家开发来描述比例。 | 是分数、比例和比率的基石。 |
| 无理数 | 不能表示为分数的数（例如，√2，π）。 | 使毕达哥拉斯学派震惊，揭示了比例的局限性。 | 在分析、几何和超越数论中居核心地位。 |
| 实数 | 所有有理数和无理数构成的一条连续线。 | 在微积分中编码来测量连续量。 | 是实分析、几何和物理的基础。 |
| 复数 | 形式为（a + bi）的数，其中（\(i = \sqrt{-1}\)）。 | 用来解决没有实根的二次方程。 | 在信号处理、量子物理和控制理论中至关重要。 |
| 虚数单位（i） | 代表√-1 的符号，使实数得以扩展。 | 16 世纪由邦贝利提出。 | 用于复分析和电气工程。 |
| 素数 | 大于 1 的整数，除了 1 和自身外没有其他因数。 | 自欧几里得以来就被研究；算术的“原子”。 | 在密码学、数论和素性检验中至关重要。 |
| 合数 | 除了 1 和自身外还有其他因数的整数。 | 与素数相反；揭示了因数结构。 | 用于因式分解算法和加密。 |
| 可除性 | 如果存在某个整数（k），使得（b = ka），则数（a）能整除（b）。 | 是模算术和 gcd 概念的基础。 | 用于计算机算术和模系统。 |
| 最大公约数（GCD） | 能整除两个整数且没有余数的最大数。 | 由欧几里得算法（约公元前 300 年）定义。 | 在简化、模算术和密码学中是基础。 |
| 最小公倍数 (LCM) | 能被两个整数整除的最小正数。 | 古代用于同步周期的工具。 | 用于日历系统、调度和离散数学。 |
| 偶数 / 奇数 | 能被 2 整除或不被 2 整除的数字。 | 是整数最早分类之一。 | 用于奇偶校验、算法和数论。 |
| 绝对值 | 数字在数轴上与零的距离。 | 无方向的量度的几何度量。 | 用于分析、优化和度量。 |
| 零 (0) | 表示无的符号，但标记位置和平衡。 | 在印度发明；通过阿拉伯学术传播。 | 是位置记数法、代数和计算的核心。 |
| 无穷大 (∞) | 无限的概念；大于任何有限数量。 | 由希腊人探索；在微积分和集合论中形式化。 | 用于极限、基数和射影几何。 |
| 负数 | 小于零的数字，代表缺席或债务。 | 在文艺复兴时期之前有争议；被笛卡尔接受。 | 用于代数、金融和坐标几何。 |
| 序数 | 表示位置或顺序的数字（第一、第二、第三…）。 | 在排名和序列中产生。 | 用于组合数学、顺序理论和编程。 |
| 实数 | 表示数量的数字（有多少）。 | 古代概念与计数和度量相关。 | 是集合基数、组合数学和数据模型的基础。 |
| 比率 | 通过除法比较两个数量的关系。 | 希腊比例理论的核心。 | 用于缩放、概率和统计学。 |
| 比例 | 两个比率的相等。 | 在相似三角形和和谐理论中的关键。 | 在物理学、金融和模型校准中居中。 |
| 分数 | 整数的一部分，表示为 ( \(\frac{p}{q}\) )。 | 在埃及和巴比伦发展；由希腊人完善。 | 用于算术、有理近似和计算。 |
| 十进制系统 | 基数为 10 的位置数制。 | 起源于印度；通过阿拉伯符号传播。 | 在测量、计算和教育中普遍使用。 |
| 位置记数法 | 价值取决于位置的体系（例如，10^n）。 | 由零启用；革命了算术。 | 是所有现代数制和计算的基础。 |
| 基数 / 阶码 | 数制中唯一数字的数量。 | 从十进制（十进制）到二进制（二进制）。 | 用于编程、编码和数字系统。 |
| 二进制数 | 以基数为 2（0,1）表示的数字。 | 由莱布尼茨复兴；计算的基石。 | 是数字逻辑和机器表示的基本。 |
| 十六进制数 | 基数为 16 的数制（0–9, A–F）。 | 在计算中引入以提高紧凑性。 | 用于内存寻址、图形和编码。 |
| 对数 | 指数运算的逆运算；将乘法转换为加法。 | 由纳皮尔发明以简化计算。 | 用于复杂性、数据尺度和增长模型。 |
| 指数/幂 | 表示重复乘法（aⁿ）。 | 自中世纪代数以来被研究。 | 用于指数增长、复利、算法。 |
| 根/根号 | 指数运算的逆（√x）。 | 在代数方程中居中。 | 用于几何学、物理学和统计学。 |
| 模数（mod） | 除法后的余数；在基数周围循环数字。 | 由高斯在模数算术中起源。 | 用于密码学、哈希函数和循环系统。 |
| 大小 | 数量的尺寸或范围。 | 在希腊数学中是哲学的；几何学意义。 | 用于向量、信号和测量理论。 |
| 标量 | 只有大小没有方向的量。 | 在向量分析中定义。 | 用于物理学、机器学习缩放和线性代数。 |
| 数量 | 可以测量或计数的任何事物。 | 是所有测量系统的根本。 | 是数学、科学和经济学中的核心概念。 |
| 连续统 | 没有间隙的集合；可无限分割。 | 是微积分和实分析的基础。 | 用于模拟连续现象。 |
| 离散 | 分离的、可数的单位；与连续相反。 | 希腊原子论 → 组合数学。 | 用于计算机科学、组合数学和概率论。 |
| 近似 | 接近但不完全等于的表示。 | 自巴比伦算术以来是实用工具。 | 用于分析、计算和工程。 |
| 有效数字 | 表示有意义精度的数字。 | 在科学测量中标准化。 | 用于误差估计、数据报告。 |
| 单位 | 测量标准（例如，米、秒）。 | 根植于贸易和几何学。 | 用于物理学、数据科学和维度分析。 |
| 维度 | 空间的方向或自由度。 | 由欧几里得定义；在代数中推广。 | 用于几何学、机器学习和物理学。 |
| 比例 | 表示现实与表示之间的比率。 | 用于地图、模型和图纸。 | 用于数据归一化和多尺度分析。 |
| 数量级 | 大小的大数近似。 | 由物理学和天文学普及。 | 用于估计、大 O 分析。 |
| 可数性 | 集合可以通过自然数枚举的性质。 | 由康托尔形式化。 | 用于集合论、拓扑学和逻辑。 |
| 不可数集 | 元素超出可数范围的集合（例如实数）。 | 由康托尔发现；令同时代人震惊。 | 用于分析、测度论。 |
| 基数 | 集合大小的度量；有限或无限。 | 由康托尔发展。 | 用于集合论、数据库设计。 |
| 范数 | 测量向量空间大小的函数。 | 来自闵可夫斯基的几何学。 | 用于优化、机器学习正则化。 |
| 度量 | 定义元素之间距离的函数。 | 在拓扑学和几何学中定义。 | 用于聚类、相似性和度量空间。 |
| 无量纲量 | 没有单位的纯数。 | 用于表示比率和常数。 | 在物理学、金融和统计学中找到。 |
| 常数 | 不变的固定值。 | 出现在方程、定律和模型中。 | π、e、c — 自然界的普遍常数。 |

### B2\. 形状与空间

*数字与形式相遇的地方——对扩展、位置和关系的研究。几何学，起源于测量土地和追踪天体的需要，发展成为结构的一种普遍语言。从线和角到流形和拓扑，这些思想揭示了世界如何占据空间以及心灵如何感知秩序。*

| 术语 | 定义 | 上下文 | 现代用法 |
| --- | --- | --- | --- |
| 点 | 空间中无大小、维度或范围的点。 | 欧几里得定义为“没有部分的东西”。 | 几何学、图论和向量空间的基础。 |
| 直线 | 向两个方向无限延伸的直线，一维图形。 | 希腊几何学的核心；用于测量和构造。 | 用于解析几何、代数曲线和线性模型。 |
| 线段 | 由两个端点界定的线的有限部分。 | 在测量和设计中实用。 | 用于 CAD、几何学和计算机图形学。 |
| 射线 | 从一个点开始并向一个方向无限延伸的线的部分。 | 用于光学和几何学。 | 在光线追踪、物理和分析几何学中找到。 |
| 平面 | 向无限延伸的平坦二维表面。 | 欧几里得几何学的核心。 | 用于向量微积分、投影和图形。 |
| 角 | 两条相交线之间旋转的度量。 | 由巴比伦人和希腊人研究。 | 用于三角学、物理和机器人技术。 |
| 顶点 | 线、边或射线相交的点。 | 几何图形的起源。 | 用于图论、多边形和计算机图形学。 |
| 平行 | 无论延伸与否，永远不会相交的线或平面。 | 由欧几里得的第五公理定义。 | 用于欧几里得几何、投影和坐标系。 |
| 垂直 | 在直角相交的线或平面。 | 是对称和结构的象征。 | 用于设计、正交性和向量空间。 |
| 多边形 | 由直线段形成的封闭图形。 | 自古美索不达米亚以来就为人所知。 | 用于几何学、建模和计算机图形学。 |
| 三角形 | 由三条边定义的最简单多边形。 | 由埃及人和希腊人研究；是三角学的基石。 | 用于三角测量、物理和有限元方法。 |
| 四边形 | 有四边的多边形（正方形、矩形、平行四边形等）。 | 在土地测量中很重要。 | 用于设计、建筑和计算几何学。 |
| 圆 | 到中心点等距离的点集。 | 在希腊思想中是完美的象征。 | 用于三角学、波动和旋转动力学。 |
| 椭圆 | 由到两个焦点的距离之和为常数的曲线。 | 由阿波罗尼奥斯研究；开普勒定律中的轨道。 | 用于天文学、统计学（置信椭圆）和设计。 |
| 抛物线 | 到焦点和准线的距离相等的曲线。 | 与抛射运动相关。 | 用于物理学、信号处理和光学。 |
| 双曲线 | 到两个焦点距离之差恒定的曲线。 | 在圆锥曲线中被研究。 | 用于相对论、导航和优化。 |
| 曲线 | 连续、平滑弯曲的线。 | 在微积分和微分几何中发展。 | 用于运动路径、建模和分析。 |
| 曲面 | 三维空间中的二维流形。 | 在几何学和拓扑学中被研究。 | 用于 CAD、流形和物理建模。 |
| 固体 | 具有体积的三维物体。 | 固体几何的基础。 | 用于 3D 建模、物理学和建筑。 |
| 多面体 | 由多边形面围成的固体。 | 由柏拉图（柏拉图固体）研究。 | 用于晶体学、几何学和 3D 渲染。 |
| 球体 | 在三维空间中与中心等距离的点集。 | 在希腊几何中是理想形状。 | 用于物理学、几何学和计算机图形学。 |
| 圆锥体 | 具有圆形底面并逐渐变尖到顶点的固体。 | 为古人所知；用于建筑。 | 用于射影几何、照明模型。 |
| 圆柱体 | 具有平行圆形底部的固体。 | 用于体积和表面积研究。 | 在工程学、几何学和流体动力学中找到。 |
| 环面 | 饼干形状的曲面；两个圆的乘积。 | 在拓扑学和复分析中被研究。 | 用于拓扑学、物理学和环面坐标。 |
| 维度 | 空间中独立方向的数目。 | 在欧几里得和后来的代数几何中定义。 | 用于数据分析、向量空间和物理学。 |
| 坐标系 | 使用数字指定位置的框架。 | 由笛卡尔引入；统一了代数和几何。 | 用于解析几何、物理学和地理信息系统。 |
| 坐标平面 | 垂直轴（x, y）的平面。 | 来自笛卡尔的《几何学》（1637 年）。 | 标准用于绘图、分析和几何。 |
| 极坐标 | 使用从原点到半径和角度的系统。 | 在圆周运动和复分析中很有用。 | 用于机器人技术、物理学和导航。 |
| 向量 | 具有大小和方向的量。 | 在 19 世纪物理学中引入。 | 用于线性代数、图形、机器学习和力学。 |
| 矩阵 | 表示线性变换的矩形数组。 | 起源于方程组。 | 用于线性代数、计算机视觉和数据科学。 |
| 变换 | 将点从一空间映射到另一空间的规则。 | 现代几何学的核心。 | 用于图形、代数和机器学习特征空间。 |
| 对称性 | 在变换（旋转、反射等）下保持不变。 | 古代美学和科学原则。 | 用于物理学、晶体学和群论。 |
| 反射 | 沿着一条线或一个平面进行镜像。 | 在光学和几何学中被研究。 | 用于图形、设计和变换。 |
| 旋转 | 围绕固定点或轴的转动。 | 几何学中的基本运动。 | 用于机器人学、动力学和群论。 |
| 平移 | 在一个方向上以固定距离移动。 | 用于欧几里得运动。 | 在几何学、运动学和群作用中找到。 |
| 缩放 | 以一定比例放大或缩小。 | 根植于比例理论。 | 用于建模、图形和归一化。 |
| 斜变换 | 线性变换和平移的组合。 | 欧几里得变换的推广。 | 用于计算机视觉、机器人和几何学。 |
| 欧几里得几何 | 基于欧几里得公理的几何。 | 曾主导 2000 年。 | 仍然是学校几何和设计的基石。 |
| 非欧几里得几何 | 违反平行公设的几何。 | 由罗巴切夫斯基、博雅依和黎曼发展。 | 用于相对论、拓扑学和现代物理学。 |
| 流形 | 局部类似于欧几里得空间的空间。 | 现代几何和物理学的核心。 | 用于广义相对论、拓扑学和机器学习嵌入。 |
| 拓扑 | 研究在连续变形下保持的性质。 | 源于欧拉的桥梁问题。 | 用于数据分析、几何学和动力学。 |
| 同胚 | 拓扑空间之间的连续、可逆映射。 | 定义拓扑等价。 | 用于拓扑学、流形和复杂系统。 |
| 度量空间 | 具有定义的距离概念的集合。 | 20 世纪分析中引入。 | 用于聚类、几何学和机器学习相似度。 |
| 测地线 | 表面上两点之间的最短路径。 | 在黎曼几何中至关重要。 | 用于导航、广义相对论和图论。 |
| 黎曼几何 | 具有弯曲空间和内积的几何。 | 由黎曼（1854 年）发展。 | 是广义相对论和现代几何的基础。 |
| 分形 | 在每个尺度上重复的自相似形状。 | 由曼德尔布罗特提出。 | 用于模拟自然、混沌和计算机艺术。 |
| 欧几里得空间 | 没有固定原点的空间，强调方向和比例。 | 向量空间的推广。 | 用于几何学、物理学和图形学。 |
| 投影几何 | 研究在投影下不变的性质。 | 源于文艺复兴时期的艺术。 | 用于视觉、透视和代数几何。 |
| 凸性 | 如果任意两点之间的线段都在集合内部，则该集合是凸的。 | 优化中的基本概念。 | 用于凸分析、机器学习和几何。 |
| 边界 | 将集合与其补集分开的边缘。 | 用于拓扑学和分析。 | 用于偏微分方程、几何和边值问题。 |
| 内部/外部 | 区域内部或外部的点。 | 在拓扑学中定义。 | 用于空间分析、几何学和微积分。 |
| 方向 | 轴或表面的有序排列。 | 在微分几何中定义。 | 用于图形学、物理学和流形。 |
| 体积 | 三维范围的度量。 | 由阿基米德研究。 | 用于积分、物理学和工程。 |
| 面积 | 二维范围的度量。 | 几何学的核心。 | 用于微积分、设计和地理信息系统。 |
| 长度 | 一维范围的度量。 | 测量的最早形式。 | 用于几何学、物理学和分析。 |
| 形状 | 对象的几何形状。 | 自古以来就有研究。 | 用于计算机视觉和模式识别。 |
| 结构 | 部分的排列和关系。 | 哲学起源。 | 用于数学、建筑和系统。 |
| 空间 | 对象和关系存在的领域。 | 从欧几里得到爱因斯坦。 | 物理学、几何和数据科学的核心。 |

### B3\. 运动 & 变化

*数学在学会运动时变得生动。从行星的转动到河流的流动和人口的增长，对变化的研究产生了微积分——变化的语法。这一系列追踪了运动、变化和转换如何将静态数量转化为动态法则。*

| 术语 | 定义 | 上下文 | 现代用法 |
| --- | --- | --- | --- |
| 变化 | 量随时间或空间的变化。 | 根植于古代对自然界循环的观察。 | 微积分、物理学和系统理论的基本。 |
| 运动 | 相对于参考点的位置连续变化。 | 由亚里士多德研究，由伽利略和牛顿革命化。 | 用于力学、机器人和动画。 |
| 速率 | 相对于另一个量的变化度量。 | 来自早期科学中的比率。 | 用于速度、增长和反应动力学。 |
| 速度 | 带方向的位移变化率。 | 由牛顿力学定义。 | 用于物理学、向量微积分和模拟。 |

### B4\. 逻辑与证明

*数学的核心在于对确定性的追求——区分真理与幻觉、必然与信仰。逻辑为推理提供结构；证明将直觉转化为知识。从亚里士多德的三段论到哥德尔的不完备性，这一系列图表展示了人类试图将思维本身形式化的尝试。*
