# 矩阵方法

> [`www.algorithm-archive.org/contents/matrix_methods/matrix_methods.html`](https://www.algorithm-archive.org/contents/matrix_methods/matrix_methods.html)

数学是许多科学和工程领域的基石，这些应用通常需要操作大型矩阵。尽管这些应用本身不是数学，但它们在所有操作中都使用数学，而操作矩阵的方法当然也是数学性的。因此，我认为我们应该探讨将矩阵转换为任何可用形式所使用的方法。大部分情况下，我们可以将数学矩阵方法分为以下几类：

1.  对角化/求特征值

1.  创建上三角和下三角矩阵

1.  解方程组

当然还有更多方法，如果我没有列出您最喜欢的算法类型之一，请不要介意。随着我们的进行，我们将继续补充！
