# 礼包包装

> 原文：[`www.algorithm-archive.org/contents/gift_wrapping/gift_wrapping.html`](https://www.algorithm-archive.org/contents/gift_wrapping/gift_wrapping.html)

如果给定一个“礼包”，这里定义为二维或三维中的随机点分布，礼包包装算法允许程序员找到其凸包——包含所有内部点的最小凸形状。这是从二维到三维跳跃导致代码变得极其复杂的情况之一。尽管如此，解决此问题的算法有着丰富的历史。

公平地说，只有 Jarvis March 被归类为**礼包包装算法**；然而，这个名字非常适合用于解决点分布凸包的算法。严格来说，这个术语并不完全准确适用于所有凸包方法。
