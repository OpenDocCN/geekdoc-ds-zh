# 计算几何

> 原文：[`www.algorithm-archive.org/contents/computational_geometry/computational_geometry.html`](https://www.algorithm-archive.org/contents/computational_geometry/computational_geometry.html)

当谈到计算数学的不同领域时，没有哪一个比计算几何给我带来更多的快乐。在某种程度上，它是几乎所有自动生成二维和三维图形的基础。如果你有时间去深入研究一些有趣的研究，我肯定会推荐你访问[arXiv.org 的计算几何部分](https://arxiv.org/list/cs.CG/recent)。随着档案的发展，我们将在这个部分添加更多内容，所以如果你想要在未来涵盖任何算法，请告诉我！
