# 决策问题

> 原文：[`www.algorithm-archive.org/contents/decision_problems/decision_problems.html`](https://www.algorithm-archive.org/contents/decision_problems/decision_problems.html)

我在这里要坦白：我不确定应该将这些问题放在哪个章节，所以目前它们被放在了*计算数学*中，但这种情况可能在不久的将来发生变化。我也不确定我们是否可以将这些问题归类为*决策问题*；然而，本节中的所有内容都与在分配适当的输出时做出明智决策相关。例如，匈牙利算法读取一系列输入（某些工作的候选人或职位）并将每个输入分配给一个输出任务，同时最小化所有输入执行其指定任务的相关成本。

总体来说，这些问题无处不在，并且它们本身绝对值得研究！
