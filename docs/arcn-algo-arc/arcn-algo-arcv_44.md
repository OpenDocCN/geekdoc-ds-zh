# 计算机图形学

> 原文：[`www.algorithm-archive.org/contents/computer_graphics/computer_graphics.html`](https://www.algorithm-archive.org/contents/computer_graphics/computer_graphics.html)

在所有计算机科学研究领域，没有哪个领域比计算机图形学对多媒体产生更直接的影响。这个子领域与计算科学明显不同，因为它关注的是真实细节的**外观**，而不是精确计算这些细节。一个计算科学家可能花费数年时间为运行在人类已知最快计算机上的软件编写代码，以模拟气候，而计算机图形学研究人员可能会应用机器学习来创建看起来足够好的流体模拟，即使对于未经训练的眼睛也是如此。最终，计算科学家将有一个图表，而计算机图形学研究人员将有一个美丽渲染的模拟。

虽然我可能把计算机图形学描绘得有点随意，但这与事实相去甚远！相反，我会争辩说，这个研究领域提供了桌面硬件目前能够支持的与真实可视化最接近的近似。许多艺术和视频游戏工作室都希望通过计算媒体讲述一个完整的故事，而这在全世界研究人员的严格努力下是根本不可能的。这就是为什么皮克斯公司雇佣研究人员，并在电影发布后积极发布他们的发现。

虽然计算机科学研究领域的边界有点模糊，但为了算法档案的目的，我们将广泛地将计算机图形学归类为任何直接应用于图像或可以表示为图像的领域。例如，卷积不被认为是计算机图形学的一部分，因为它们在计算机科学研究的所有领域都得到了广泛应用；然而，Canny 边缘检测将会。我们还将涵盖广泛的应用，这些应用用于渲染高分辨率图形和计算艺术。

就像算法档案的所有部分一样，这是一个正在进行中的工作，可能会发生变化，所以请随时告诉我您的想法！
