# 数学背景

> 原文：[`www.algorithm-archive.org/contents/mathematical_background/mathematical_background.html`](https://www.algorithm-archive.org/contents/mathematical_background/mathematical_background.html)

无论你问谁，编程至少需要一点数学。话虽如此，对于大多数程序员来说，这并不需要*太多*。在很大程度上，根据你的专业，你可能不会看到太多的微积分或微分方程。说实话，你可能可以用你在高中学到的知识应付过去。

然而，这是一本关于算法的书，算法有时需要更深入地理解数学。本节试图提供您理解某些算法所需的数学基础。随着我们添加新的算法并需要新的数学工具，我们将把它们添加到这一节中。

对于需要特定领域知识的算法类别，如量子模拟或生物信息学，这一规则将有一个显著的例外。在这些情况下，我们将把数学方法放在更相关的部分。
