# 物理求解器

> 原文：[`www.algorithm-archive.org/contents/physics_solvers/physics_solvers.html`](https://www.algorithm-archive.org/contents/physics_solvers/physics_solvers.html)

有一些算法是专门为解决特定物理系统而创建的。例如，运动方程可以用 Verlet 积分求解，也可以用更通用的微分方程求解器求解。在本节中，我们将放置所有特定于物理且不能用于任何其他领域的算法和方法。值得注意的是，在整个档案中使用的许多算法也可以用于求解物理方程，物理比许多其他科学领域（如生物学或化学）更容易采用计算方法。

在未来，我们可能还会放置几个独特的物理系统以及解决这些系统的不同方法。例如，有解决薛定谔方程、热方程、运动方程以及一些其他独特于物理的系统的许多不同方法。以某种方式解决某些物理系统有其优点和缺点。
