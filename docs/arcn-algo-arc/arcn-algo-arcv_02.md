# 简介

> 原文：[`www.algorithm-archive.org/contents/introduction/introduction.html`](https://www.algorithm-archive.org/contents/introduction/introduction.html)

人们的学习方式各不相同，这一点毫无疑问。就我个人而言，我从未能够跟上讲座的节奏。无论主题多么有趣，老师或教授要么讲得太慢，要么讲得太快，在两种情况下，我都会失去课外学习和学习的动力。回顾我过去所有的讲座，这真是一件遗憾的事情。我真心热爱学习！事实上，如今没有什么比翻阅[ArXiv](https://arxiv.org/)，寻找物理学和计算机科学中最新的、最优秀的算法更让我感到快乐了。当然，我无法完全理解它们，但有时候，完全被压倒并一点一点地解决问题是件好事。最终，现代研究是一系列有趣的（或大或小）的谜题，等待我们去解决；然而，我发现许多论文对于大多数没有计算机科学或物理学背景的人来说太难了。正因为如此，它们被公众忽视了。我们生活在一个几乎整个人类知识都触手可及的时代，然而人们却对推动我们理解前进的独特思想和挑战毫无察觉。

我想这就是这项工作的意义所在。在过去几年里，我一直在收集那些难以通过博客文章、YouTube 或 Twitch 直播有意义地表达的知识碎片。例如，YouTube 是一个介绍算法一般概念的绝佳地方；然而，我个人认为展示一个工作示例并不是一个好的选择。Twitch 非常适合展示一切，但第一次编程一个算法需要花费 5 个小时，这通常不是每个人认为有趣的事情。博客文章对于这类材料来说效果很好；然而，它们缺乏我倾向于享受的全面性。那么，剩下的是什么？嗯，我想那可能就是一本书，这是一个非常令人兴奋的前景！

看看吧，当我年轻的时候，可能是在初中时期，我非常想成为一名作家。我尽可能地利用空闲时间写关于英雄和恶棍创造自己的世界和重新定义现实的奇幻故事。这（现在仍然是）我的一项爱好。在许多方面，我对讲故事的热情体现在我对科学和数学的热爱上。在我的心中，每一个发现都是自己的故事——一个等待被理解的冒险！对我来说，现在有机会将我最喜欢的两个爱好结合起来：写作和学习，这真是太不可思议了。

现在，为了明确起见，我并不是一个计算机科学大师。实际上，我的学位是物理学，在写作的时候，我正在攻读量子系统博士学位；然而，我真的非常喜欢学习算法。大约一年或两年前，我开始了一个专注于计算物理学的 Twitch 直播。几乎每天早上 5 点我都会起床，编写我之前从未编写过的代码。这非常有趣，通过与社区的合作，我们学到了很多。此外，我发现我的编程能力有了显著提高，并学会了看到业余程序员社区的力量。

这本书是那个想法的自然延伸——一个社区努力尽可能多地了解尽可能多的算法。正因为如此，我觉得我应该尽可能诚实地介绍这本书的本质。如前所述，我在计算机科学方面没有接受过传统训练，所以大部分时间，我将会和读者一起学习。我的希望是在提高自己对计算机科学理解的同时，也为那些希望提高自己理解的人提供一份全面的指南。

因此，我犹豫着是否称这本书为“教科书”，更愿意将其视为一种类型的实验笔记。它将是一个充满计算机科学和物理学中酷炫想法的档案，我希望它们本身就很有趣。我的希望是这本书完美地平衡了全面性和乐趣，并且对于任何阶段的业余编程爱好者来说都是一本愉快的读物。

沿着社区的主题，这本书可以在[GitHub](https://github.com/algorithm-archivists/algorithm-archive)和[GitHub pages](https://www.algorithm-archive.org)上免费获取。请随意阅读/浏览那里的内容。如果您有任何问题/评论/关注点，请告诉我。这本书是一个社区项目，只有当有社区一起工作时才能发挥作用！

在开始大量正文之前，我想强调的是，我希望这本书是“语言无关的”。这基本上意味着我会在正文中使用伪代码，但在附录中提供任何您想要的源代码。虽然这听起来可能有些反直觉，但我认为这为作者和我以及社区以任何他们喜欢的方式开发自己的代码提供了最大的灵活性。事实上，如果您愿意，请随意实现文中提到的任何算法的自己的版本。如果您提交它并且它通过了我的*嗯*严格测试，那么我可以在每个章节的末尾添加您的贡献！

所以，我想现在就到这里吧。因为这本书可以在网上免费获取，随着我们深入挖掘古老的算法档案，我可能会对内容进行大量的修改。
