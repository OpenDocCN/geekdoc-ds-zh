## 关于本书

这本书是为了在计算机课程中使用而编写的，该课程教授统计学或数据科学系的研究生 R 和 Python。这本书是为那些不一定具备编写代码经验的学生的。

+   如果你使用它们来分析数据，R 和 Python 在相当相似的方式上做了很多相同的事情，所以有时教一种语言后再教另一种语言，或者选择其中一种而放弃另一种，都没有太大的意义。并排的方法有助于教学和学习更多，节省时间，并加强共享概念和关键差异。

+   这本书并没有详细描述统计建模技术，但许多练习会要求学生实现它们。这些练习不会假设学生具备数学知识。目标是测试编程能力，而且额外的益处是，这些工作可能在他们的职业生涯中后来会变得有用。

+   这本书是为有志成为数据科学家的人编写的，不一定是有志成为软件开发者的人。我为什么要区分这一点？例如，在讨论不同类型时，我不会深入讨论数据结构。相反，我会讨论不同类型在哪些应用中最有用的例子。

+   通常来说，章节应该按顺序阅读，但偶尔跳过一些内容可能会有好处。后面的章节可能假设读者对前面的章节有所了解。此外，在章节的某个部分，有时会先讨论 Python 中的特定主题，比如，然后再讨论 R 中的相同主题。在这种情况下，后面的部分通常会假设先阅读了前面的部分。

## 习惯用法

有时 R 和 Python 代码看起来非常相似，甚至完全相同。这就是为什么我通常将 R 和 Python 代码分开成不同的部分。然而，有时我并不这样做，所以每当需要防止混淆时，我会在注释中提醒你正在使用哪种语言（更多关于注释的内容在 1.2）。

```py
# in python
print('hello world')
## hello world
```

```py
# in R
print('hello world')
## [1] "hello world"
```

## 安装所需的软件

要开始，你必须安装 R 和 Python。安装过程取决于你的机器类型（例如，你的机器运行的是哪种操作系统，你的处理器是 32 位还是 64 位等）。

下面，我建议使用 RStudio 运行 R，使用 Anaconda 运行 Python，并提供一些有用的链接。我建议分别下载这两个捆绑包；然而，我应该指出，下面的建议并不是唯一的安装方法。例如：- 可以运行 R 和 Python 而不下载 RStudio 或 Anaconda，- 可以在 Anaconda 中安装 RStudio，- 可以在 RStudio 中运行 Python，- 可以在由 Anaconda 管理的 RStudio 中运行 Python，等等，- 选项和程序很可能随着时间的推移而发生变化。

如果讲师希望使用替代策略，他们应该验证 Python 的版本是否`>=3.6`，R 的版本是否`>=4.0.0`。如果是这样，这本书中的所有代码都应该能够运行。

### 安装 R（和 RStudio）

建议您安装 R 和**RStudio Desktop**。**RStudio Desktop**是一个图形用户界面，拥有许多工具，使编写 R 代码变得更加容易和有趣。

从综合 R 档案网络（CRAN）安装 R。您可以通过点击[这里](https://cran.r-project.org/)获取您特定机器的安装说明。

您可以直接从[公司网站](https://www.rstudio.com/products/rstudio/download/#download)获取 RStudio Desktop。

### 通过安装 Anaconda 安装 Python

建议您安装**Anaconda**，这是一个包管理器、环境管理器和包含许多第三方开源软件包的 Python 发行版。它也为我们提供了一个图形用户界面，就像 RStudio 一样。您可以通过点击[这里](https://docs.anaconda.com/anaconda/install/#)获取您特定机器和操作系统的安装说明。
