# 关于作者

> [作者介绍](https://bruno.nicenboim.me/bayescogsci/about-the-authors.html)

Bruno Nicenboim ([`bruno.nicenboim.me`](https://bruno.nicenboim.me)) 是荷兰蒂尔堡大学认知科学和人工智能系的一名助理教授，在那里他是计算心理语言学研究线的首席研究员（PI）。他最初在阿根廷罗萨里奥国立大学学习电子工程，然后转向人文学科，并在以色列度过了八年，在那里他在特拉维夫大学完成了社会学和语言学学士学位以及语言学硕士学位。在此期间，他还曾在几家 IT 公司工作。之后，他搬到了德国，在波茨坦大学完成了认知科学博士学业，并在波茨坦大学担任了两年博士后研究员。他的研究兴趣包括贝叶斯方法、计算认知建模、句子理解、记忆过程、决策和预测处理。他经常在研讨会和暑期学校中教授贝叶斯数据分析短期课程，并在蒂尔堡大学的认知科学和人工智能以及数据科学与社会硕士项目中教授贝叶斯建模和统计学课程。

Daniel J. Schad ([`danielschad.github.io/`](https://danielschad.github.io/)) 是德国波茨坦 HMU 健康与医学大学心理学系定量方法学教授。他在德国波茨坦大学和美国密歇根大学安娜堡分校学习心理学。他在波茨坦大学获得了认知和数学心理学博士学位，研究眼动控制的计算模型和无意识阅读。随后，他在德国柏林夏里特-大学医学中心（部分时间也在波茨坦大学）的计算机精神病学领域进行了五年的博士后研究，在瑞士苏黎世联邦理工学院和英国伦敦大学学院进行了研究访问，研究无模型和基于模型的决策以及酒精依赖中的巴甫洛夫式-工具性转移，以及巴甫洛夫式条件反射背后的认知和大脑机制。他在波茨坦大学担任博士后研究员，研究认知科学中的定量方法，包括对比、显著性检验的性质、贝叶斯工作流程和贝叶斯因子分析，并在蒂尔堡大学认知科学和人工智能系担任助理教授。

Shravan Vasishth ([`vasishth.github.io`](http://vasishth.github.io)) 是德国波茨坦大学心理语言学的教授。他担任心理语言学和神经语言学（语言处理）的教席。他在印度新德里贾瓦哈拉尔·尼赫鲁大学完成日语学士学位后，在日本大阪度过了总共五年的时间。在日本期间，他在大阪外国语大学学习日语，在大阪大学进行研究，并在大阪的一家专利律师事务所担任内部翻译。他在美国俄亥俄州立大学完成了计算机与信息科学硕士学位（2000-2002）和语言学博士学位（1997-2002），以及英国谢菲尔德大学数学与统计学学院的统计学硕士学位（2011-2015）。他是特许统计师（英国皇家统计学会；ID 128307）。他是开放获取期刊 Glossa: Psycholinguistics 的编委会成员。他的研究专注于无损伤和损伤人群中的句子处理计算建模，以及将数学、计算、实验和统计方法（尤其是贝叶斯方法）应用于语言学和心理学。他创建了一个年度夏季学校，统计方法在语言学和心理学中的应用（SMLP），始于 2017 年：vasishth.github.io/smlp。他定期教授关于统计数据分析的短期课程（贝叶斯和频率主义方法）。
