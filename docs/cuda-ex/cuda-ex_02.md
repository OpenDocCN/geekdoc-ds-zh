## 致谢

有人说，写一本技术书籍需要一个“村庄”的帮助，《*CUDA by Example*》也不例外。作者们要感谢许多人，在这里我们特别感谢以下几位。

Ian Buck，NVIDIA 的 GPU 计算软件高级总监，在本书的每个开发阶段都提供了巨大的帮助，从推动这个想法到管理许多细节。我们还要感谢 Tim Murray，这位总是面带微笑的审稿人，正是他让本书在技术准确性和可读性方面达到了一个基本的标准。此外，特别感谢我们的设计师 Darwin Tat，他在极为紧张的时间表下创作了精彩的封面艺术和图表。最后，我们还要非常感谢 John Park，他帮助本书顺利完成了出版过程中需要的法律程序。

如果没有 Addison-Wesley 的工作人员的帮助，这本书依然只会是作者们心中的一个梦想。Peter Gordon、Kim Boedigheimer 和 Julie Nahil 展现了无尽的耐心和专业精神，真正让本书的出版过程变得轻松顺畅。此外，Molly Sharp 的制作工作和 Kim Wimpsett 的校对工作，彻底改变了这篇充满错误的文档，使它最终变成了你现在阅读的这本书。

本书中的一些内容在没有其他贡献者帮助的情况下是不可能被包含的。具体而言，Nadeem Mohammad 在我们展示的 Cuda 案例研究的研究中发挥了重要作用，而 Nathan Whitehead 则慷慨提供了代码，我们将其纳入了书中的示例中。

如果我们不感谢那些早期阅读过这篇文章并提供宝贵反馈的人，我们将感到遗憾，其中包括 Genevieve Breed 和 Kurt Wall。在开发《*CUDA by Example*》内容的过程中，许多 NVIDIA 软件工程师提供了不可或缺的技术支持，包括 Mark Hairgrove，他仔细审阅了本书，发现了各种不一致之处——无论是技术问题、排版问题，还是语法问题。Steve Hines、Nicholas Wilt 和 Stephen Jones 在 CUDA API 的特定部分提供了咨询，帮助阐明了作者可能忽视的细微差别。同时，感谢 Randima Fernando 在项目启动阶段的帮助，以及感谢 Michael Schidlowsky 在书中提到 Jason。

在感谢部分，怎能少了对父母和兄弟姐妹的衷心感谢呢？在这里，我们要感谢我们的家人，他们一直陪伴着我们，支持我们，让这一切成为可能。在此，我们特别感谢慈爱的父母，Edward 和 Kathleen Kandrot 以及 Stephen 和 Helen Sanders。还要感谢我们的兄弟，Kenneth Kandrot 和 Corey Sanders。感谢你们对我们的坚定支持。
