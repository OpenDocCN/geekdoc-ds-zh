## 前言

本书展示了如何通过利用计算机的图形处理单元（GPU）来编写适用于广泛应用程序的高性能软件。虽然最初设计用于在显示器上渲染计算机图形（并且仍然用于这一目的），但 GPU 在科学、工程、金融等领域越来越多地被应用于同样要求高的程序。我们将这些解决非图形领域问题的 GPU 程序统称为*通用程序*。幸运的是，虽然你需要有一定的 C 或 C++ 编程经验才能从本书中受益，但你不需要了解计算机图形学，完全不需要！GPU 编程只为你提供了一个基于你现有编程技能进行“强大构建”的机会。

要编写程序以使 NVIDIA GPU 执行通用计算任务，您需要了解什么是 CUDA。NVIDIA GPU 基于被称为*CUDA 架构*的技术。你可以把 CUDA 架构看作是 NVIDIA 构建可以执行*传统图形渲染任务*和*通用任务*的 GPU 的方案。为了编程 CUDA GPU，我们将使用一种叫做*CUDA C*的语言。正如你将在本书的早期看到的，CUDA C 本质上是 C 语言，通过一些扩展来支持编程大规模并行计算机，比如 NVIDIA GPU。

我们将*CUDA by Example*定位为面向有经验的 C 或 C++ 程序员，他们对 C 语言有足够的熟悉度，能够舒适地阅读和编写 C 代码。本书建立在你对 C 语言的经验基础上，旨在作为一个示例驱动的“快速入门”指南，帮助你使用 NVIDIA 的 CUDA C 编程语言。你完全不需要做过大规模的软件架构设计，编写过 C 编译器或操作系统内核，或知道 ANSI C 标准的所有细节。然而，我们不会花时间回顾 C 语法或常见的 C 库函数，如 `malloc()` 或 `memcpy()`，因此我们假设你已经对这些主题有一定了解。

你将会接触到一些可以视为通用并行编程范式的技术，尽管本书并不旨在教授通用并行编程技术。此外，虽然我们将几乎涵盖所有 CUDA API 的内容，但本书并不作为一个详尽的 API 参考，也不会深入讨论你可以用来开发 CUDA C 软件的所有工具。因此，我们强烈建议本书与 NVIDIA 免费提供的文档一起使用，特别是*NVIDIA CUDA 编程指南*和*NVIDIA CUDA 最佳实践指南*。不过不要担心收集所有这些文档，因为我们会带你完成所需的所有步骤。

所有必需的 NVIDIA 软件可以通过链接从[`developer.nvidia.com/object/gpucomputing.html`](http://developer.nvidia.com/object/gpucomputing.html)找到。本书第二章讨论了开始编写 CUDA C 程序时绝对需要的组件。由于本书旨在通过示例教学，因此包含了大量的代码示例。这些代码可以从[`developer.nvidia.com/object/cuda-by-example.html`](http://developer.nvidia.com/object/cuda-by-example.html)下载。

不再赘述，编程 NVIDIA GPU 使用 CUDA C 的世界等待着你！
