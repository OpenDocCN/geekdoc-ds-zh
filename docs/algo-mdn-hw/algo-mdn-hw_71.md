# 算法案例研究

> 原文：[`en.algorithmica.org/hpc/algorithms/`](https://en.algorithmica.org/hpc/algorithms/)

当你试图解释一个复杂的概念时，给出一个非常简单且最小化的例子来阐述它通常是一个好主意。这就是为什么在这本书中，你会看到大约有十种不同的计算数组总和的方法，每种方法都突出了某种 CPU 特性。

但本书的主要目的并不是为了学习计算机架构而学习它，而是为了获得软件优化的实际技能。接下来的两章旨在帮助你实现这一目标，因为它们包含了各种算法的详细案例研究，这些算法比数组求和要难以优化得多。
