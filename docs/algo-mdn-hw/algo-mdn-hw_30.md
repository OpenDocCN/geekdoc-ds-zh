# 算术

> 原文：[`en.algorithmica.org/hpc/arithmetic/`](https://en.algorithmica.org/hpc/arithmetic/)

正如我们在本书中反复演示的那样，了解指令集的暗角可以非常有成效，尤其是在像 x86 这样的 CISC 平台的情况下，它目前有[1000 到 4000 条](https://stefanheule.com/blog/how-many-x86-64-instructions-are-there-anyway/)不同的指令，具体取决于你如何计数。

大多数这些指令都与算术相关，要有效地使用它们来优化算术运算，需要大量的知识、技能和创造力。因此，在本章中，我们将讨论数字表示及其在数值算法中的应用。
