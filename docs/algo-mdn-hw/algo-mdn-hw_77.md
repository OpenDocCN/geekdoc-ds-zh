# 数据结构案例研究

> 原文：[`en.algorithmica.org/hpc/data-structures/`](https://en.algorithmica.org/hpc/data-structures/)

优化数据结构与优化算法不同，因为数据结构问题有更多的维度：你可能正在优化*吞吐量*，*延迟*，*内存使用*，或者这些因素的任何组合——当你需要处理*多种*查询类型并考虑多种查询分布时，这种复杂性会呈指数级增长。

这使得简单地定义基准变得非常困难，更不用说实际的实现了。在本章中，我们将尝试导航所有这些复杂性，并学习如何通过广泛的案例研究设计高效的数据结构。

强烈建议简要回顾 CPU 缓存系统。
