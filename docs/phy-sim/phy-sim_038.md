# 非穿透距离障碍

接触建模是确保固体不与障碍物或自身相交的关键方面。这个主题在之前的章节中简要提及。在本讲中，我们将深入探讨增量势能接触（IPC）方法框架内非穿透的具体细节。我们的重点将放在一个简化但重要的场景上：具有封闭边界的固体与障碍物之间的接触。这种特定的关注点使我们能够在可控环境中彻底探索 IPC 方法的力学和原理。
