# 摩擦

> 原文：[`phys-sim-book.github.io/lec2.4-friction.html`](https://phys-sim-book.github.io/lec2.4-friction.html)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.4/dist/katex.min.css">

摩擦力是涉及运动的物理相互作用中的关键要素，通常对模拟结果有显著影响。因此，其精确建模对于现实和可靠的模拟至关重要。请参见右侧的图 2.3.1，以演示需要精确表示摩擦力的场景。

摩擦力最广泛采用的模型之一是**库仑摩擦模型**。该模型基于**最大耗散原理（MDP**），有效地捕捉了静态摩擦和动态摩擦之间的非光滑过渡。静摩擦是阻止物体开始运动的力，而动态摩擦，或称为运动摩擦，则反对运动物体的运动。库仑摩擦模型准确地描述了这两种摩擦类型之间的临界过渡。

在标准材料点方法（MPM）中，摩擦力本质上是通过网格进行建模的。然而，这种方法有其缺点，特别是摩擦系数过大且不可控。

对于有限元方法（FEM），摩擦力可以通过增量势能接触（IPC）模型中的近似势能更真实、更可控地表示。这很好地符合我们的优化时间积分框架。通过使用势能来近似摩擦力，我们不仅保持了模拟的鲁棒性，还获得了对摩擦模型精度的控制。

在后续的讲座中，我们将深入探讨在 IPC 模型中表示摩擦力所采用的具体技术和方法，以及它们在提高模拟精度和现实性方面的作用。
