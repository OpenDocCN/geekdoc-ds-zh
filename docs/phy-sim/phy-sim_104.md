# 摘要

> 原文：[`phys-sim-book.github.io/lec20.4-summary.html`](https://phys-sim-book.github.io/lec20.4-summary.html)



我们将离散弱形式（方程 (19.1)）与滤波线搜索中的实现连接起来，用于边界条件和接触。此外，我们还推导了二维离散表面的自接触，这将在下一节课中实现。

推导遵循一致的方法：首先，将全局积分重写为局部元素积分的和，然后使用某些求积规则近似或解析地评估局部积分。

在本节课中，我们没有明确讨论摩擦，因为在连续设置中其力的定义已在边界条件和摩擦接触中介绍。其积分近似可以类似于正常接触力（详见案例研究：2D 摩擦自接触）。

在推导过程中，我们还观察到，我们从强形式到优化时间积分实现的路线，即：

强形式→弱形式→离散弱形式→有限元近似→优化时间积分

并非唯一。我们可以直接写出势能的连续形式，然后进行空间离散化和近似以获得节点力。对这种方法感兴趣的读者可以参考拉格朗日力学或哈密顿力学。
