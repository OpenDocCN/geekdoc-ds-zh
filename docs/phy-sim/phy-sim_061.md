# 运动学理论

> 原文：[`phys-sim-book.github.io/lec12-kinematics.html`](https://phys-sim-book.github.io/lec12-kinematics.html)



在之前的案例研究中，我们依靠质量-弹簧模型来模拟固体的弹性行为。该模型通过在不同方向上连接多个弹簧来近似 2D 和 3D 弹性，每个弹簧只对拉伸和压缩做出反应。然而，这种简单的近似往往无法捕捉现实世界现象的复杂性。从这次讲座开始，我们将深入研究变形的数学描述，并介绍一种更严格的方法来模拟连续介质的弹性。

当讨论连续介质或连续介质力学时，我们基于**连续性假设**进行操作。这种观点将材料——无论是固体、液体还是气体——视为连续实体，避免了需要考虑分子和原子之间的微观相互作用。这个假设不仅在工程和图形应用中实用，而且在日常生活中也很普遍。

在图形模拟中，连续性假设适用于广泛的各种材料，包括可变形物体（既有弹性的也有塑性的）、肌肉、肉体、布料、头发、液体、烟雾、气体以及沙子、雪、泥和土壤等颗粒材料。在连续介质力学中，密度、速度和力等属性被定义为位置的连续函数。我们在离散空间和时间部分探讨了它们的离散对应物。

基于牛顿第二定律的运动方程在空间域内求解，并随时间演化以模拟这些材料的动态行为。
