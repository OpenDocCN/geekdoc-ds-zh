# 摘要

> 原文：[`phys-sim-book.github.io/lec22.4-summary.html`](https://phys-sim-book.github.io/lec22.4-summary.html)



我们在二维空间中基于点-边对的无符号距离平方实现了半隐式摩擦，并将其纳入时间步优化中。

我们首先通过半隐式时间离散化和动态-静态摩擦过渡的平滑近似，使摩擦力在连续设置中可积。近似摩擦势的空间离散化遵循与障碍势相似的方法。

接着，我们检查了点-边对中最近点的法向力大小 λ、法向方向 n 和重心坐标 r 的计算。这些值在每个时间步的开始计算，并在优化过程中保持不变。需要注意的是，用于摩擦的点-边对集合在每个优化过程中也是恒定的，并且与用于障碍物的集合不同。

最后，我们实现了离散摩擦势及其导数的计算。我们使用了相对速度 v^k 作为中间变量，并应用链式法则来组织计算。

到目前为止，我们已经涵盖了具有无逆弹性动力学和无穿透摩擦自接触的二维固体模拟的理论和实践方面。接下来，我们将探讨将这些概念扩展到三维所需的额外步骤！
