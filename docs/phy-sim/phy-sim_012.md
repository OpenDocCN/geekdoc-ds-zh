为了准确模拟固体，确保它们不发生穿透是至关重要的，如图下所示（左侧）。一种有效的方法是强制执行**CFL**（Courant-Friedrichs-Lewy 条件）的时间步长上限，尤其是在 MPM 等方法中。在有限元方法（FEM）中，这需要精确建模接触力。然而，准确建模接触是一个挑战。接触本质上是一个非光滑过程，当固体接触时突然发生。没有势能公式能够准确描述这一现象。

**图 2.3.1（接触和摩擦的模拟示例）。** 在左侧，一个引人入胜的模拟展示了四个角色跌入漏斗并被移动的平面挤出。在此复杂交互过程中没有任何穿透，执行无懈可击，突显了所采用模型的精度。在右侧，我们看到一个经典桌布魔术的模拟，以不同的速度执行。在这个模拟中，尤其是摩擦的准确描绘，当布被拉走而不打扰桌面设置时变得明显——这与现实生活中所期望的相呼应。这些模拟展示了当代计算模型在模拟接触方面的惊人能力和精度，生动有趣地将抽象的物理概念呈现出来。

在实际应用中，确定两个物体是否发生碰撞通常涉及视觉和心智上评估它们的接近程度。当它们之间的距离不为零时，这表明空间仍然存在，没有发生碰撞。这个概念在计算环境中建模物体之间的交互时至关重要。

为了避免碰撞或穿透，我们可以确保移动物体表面的距离永远不会减少到零。这种方法在计算模拟的时间积分问题中特别有用。我们使用不等式约束来模拟这种场景，结合边界条件，将我们的时间积分问题表述如下：$ x_{min} \in E(x) \text{s.t.} Ax=b \text{ and } \forall k, c_k(x) \geq \epsilon \text{.} (2.3.1) $ 这里，$ c_k $ 衡量固体表面特定区域对之间的距离，而 $ \epsilon \rightarrow 0 $ 是一个非常小的正数，以确保 $ c_k(x) $ 保持严格正值。

在方程 (2.3.1) 问题的局部极小值处，我们遵循 Karush-Kuhn-Tucker (KKT) 条件，如下所示：∇E(x)−ATλ−k∑​γk​∇ck​(x)=0,Ax=b,∀k, ck​(x)−ϵ≥0, γk​≥0, γk​(ck​(x)−ϵ)=0。（2.3.2）在这个条件中，$ \gamma_k $ 是约束 $ c_k(x) \geq \epsilon $ 的拉格朗日乘子。为了分解它，$ \nabla c_k(x) $ 指向接触对 $ k $ 的接触力方向。这个方向与由 $ \gamma_k $ 表示的量级的组合给出了该点的实际接触力。

> ***备注 2.3.1（互补松弛条件）***。互补松弛条件 $ \gamma_k (c_k(x) - \epsilon) = 0 $ 在确保只有当固体接触时（$ c_k(x) = \epsilon $）才存在接触力（$ \gamma_k \neq 0 $）中起着关键作用。相反，当固体未接触时（$ c_k(x) > \epsilon $），则不应存在接触力（$ \gamma_k = 0 $）。
> 
> ****定义 2.3.1（活动集）****。在定义不等式约束的优化问题中，\[\forall k, \ c_k(x) \geq 0,\] **活动集**定义为 \[\{ l \ | \ c_l(x^*) = 0 \}.\] 这里，$ x^* $ 是该问题的局部最优解。
> 
> ***备注 2.3.2（组合困难）***。互补松弛条件表明，只有活动集中的约束会在解处表现出非零的拉格朗日乘子 $ \gamma_k $。这表明，与等式约束不同，不等式约束不仅需要求解拉格朗日乘子的值，还需要确定哪些 $ \gamma_k $ 应该设置为 $ 0 $。这提出了组合困难。

有许多技术可用于解决具有不等式约束的优化问题。每种方法都引入了一种独特的方法，有效地针对问题的各个方面。

+   **原对偶方法**：这类方法同时解决原问题（原始优化问题）及其对偶问题。对偶问题通常为原问题的解提供有价值的见解，使得这种方法很有吸引力。这些方法是迭代的，通过利用原问题和对偶问题之间的关系来改进初始解。然而，设计和实现原对偶算法可能很复杂，需要仔细平衡两种问题类型。虽然有效，但这些方法对于复杂、高维问题可能既不高效也不简单。

+   **投影最速下降法**：这是经典最速下降法的一种改进，这些方法处理约束条件。在每次迭代中，算法沿着最速下降方向移动，如果由于约束条件而偏离，则将其投影回可行集。这种方法简单直接，因此很受欢迎，但它可能在收敛缓慢的病态问题或难以投影到其上的约束条件下遇到困难。

+   **内点法**：也称为障碍法，这些技术引入了一个惩罚不可行解的障碍函数，从而将解引导到可行区域的内部。这种方法有效地将一个约束问题转化为一个无约束问题，可以使用传统技术求解。然而，障碍函数的选择对方法的性能有重大影响。虽然对于某些问题类型来说效率很高，但这些方法在可行区域难以定义或缺乏简单内部结构的问题上可能会失败。

虽然每种方法都有其自身的优点和缺点，但我们的主要关注点将是一个稳健且精确的接触建模方法，称为**增量势能接触（IPC）**。IPC 通过用平滑的势能近似接触过程来区别于其他方法。这种转换有效地将问题转化为一个无约束问题，从而便于应用各种高效且稳健的优化技术。IPC 的一个关键特性是它能够控制相对于非光滑公式的近似误差在预定界限内。这一特性为方法增加了一层稳健性和可靠性，使其成为解决当前问题的特别有前景的方法。
