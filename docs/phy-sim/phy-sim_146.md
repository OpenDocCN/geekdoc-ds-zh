# 屈服条件和返回映射

为了在离散的时间和空间中实施屈服条件和流动规则，我们应用一种称为**返回映射**的程序。这是一个基于投影的算法，它修改弹性变形梯度，使得产生的应力满足屈服条件。

### 返回映射

令 Ftr 表示**试弹性变形梯度**，在粒子状态更新之后但在塑性施加之前获得。如果从 Ftr 计算出的应力位于屈服面**内部**，则不需要校正，因为变形完全是弹性的。

如果应力位于屈服面**外部**，我们应用返回映射来校正试弹性状态，将 Ftr 产生的应力投影回屈服面，遵循塑性流动规则指定的方向。我们将返回映射的结果表示为：

FEn+1 = Z(Ftr),

其中 Z(⋅) 是**返回映射算子**，它根据屈服面几何形状和材料模型对弹性预测进行校正。

在实践中，返回映射的实现方式可以是解析的（对于简单的屈服准则，如冯·米塞斯或德鲁克-普拉格）或者通过数值求解器迭代进行，当屈服面复杂时。

> **示例 28.2.1（冯·米塞斯屈服准则，对数应变公式）。**
> 
> 在基于对数应变的塑性中，返回映射在对角化奇异值分解（SVD）空间中进行。令 Fn+1 = UΣVT，令 σpn 表示存储的对数塑性应变。试弹性拉伸为：Σtr = Σ⋅(exp(σpn))−1, ϵtr = log(Σtr)。计算偏斜对数应变：ϵdev = ϵtr − d1 tr(ϵtr)I。如果 ∥ϵdev∥>2/3 ϵY，则执行径向返回：ϵdevn+1 = (1−∥ϵdev∥∥ϵdev∥−2/3 ϵY)ϵdev。校正后的弹性对数应变为：ϵlogn+1 = ϵdevn+1 + d1 tr(ϵtr)I。然后更新：ΣEn+1 = exp(ϵlogn+1), σpn+1 = log(Σ)−log(ΣEn+1)，并重建弹性变形：FEn+1 = UΣEn+1 VT。
