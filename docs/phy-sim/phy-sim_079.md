# 摘要

> 原文：[`phys-sim-book.github.io/lec15.4-summary.html`](https://phys-sim-book.github.io/lec15.4-summary.html)

`<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.4/dist/katex.min.css">`

我们已经成功实现了一个无反演的 2D 弹性模拟，通过使用线性三角形元素对 Neo-Hookean 模型进行离散化。

通过在每个三角形内保持线性变化的位移场，我们可以直接使用顶点的材料和世界空间坐标来计算每个三角形的恒定变形梯度 F。这种基础设置通过应用链式法则，便于计算 Neo-Hookean 能量及其关于 x 的梯度和对 Hessian，这对于之前讲座中讨论的基于优化的时间积分是必不可少的。

为了确保模拟既无穿透也无反演，我们采用了之前描述的类似策略：线搜索的初始步长通过为每个三角形求解一个二次方程来确定。该方程计算出一个临界步长，该步长将三角形的体积减少 90%。然后，使用所有三角形中这些临界步长中最小的一个来初始化线搜索，确保对非穿透和非反演的鲁棒性。

在即将到来的章节中，我们将深入探讨超弹性固体的控制方程的推导过程，并对每一步进行详细解释，以进一步巩固理解。
