# 离散空间和时间

> 原文：[`phys-sim-book.github.io/lec1-discrete_space_time.html`](https://phys-sim-book.github.io/lec1-discrete_space_time.html)



在这次讲座中，我们探讨了对可变形固体的模拟，目的是开发一个离散的、可计算的问题。主要目标是介绍这个问题内在的抽象代数概念。我们采用自顶向下的架构视图来处理弹性模拟，将数学建模置于首位。

经典弹性固体物理的研究在很大程度上围绕着偏微分方程（PDEs）。在连续介质力学和有限元分析文献中，通常的做法是首先推导这些 PDEs 的连续形式，详细阐述每个项的来源，然后再将其适应到离散编程语言中。通常，这种适应出现在后面的章节中，给读者一种期待感。

然而，这本书采取了不同的路径。它根据需要将连续介质力学和偏微分方程（PDEs）融入讨论中，均匀地分配这些主题，以避免让读者感到不知所措。这种方法逐步将理论与实践联系起来，增强了对理论的理解。

我们早期引入主要问题公式化，提供其数值解的概述。这给读者提供了一个初步的全面视角，激发好奇心，并激励在后续章节中进行更深入的探索。这种策略使学习过程更加流畅和直观，帮助读者轻松地将复杂概念联系起来，并迅速掌握主题的核心。

我们的目的是提供一个全面、深入且引人入胜的可变形固体模拟探索，对领域内的学生和资深研究人员都极具价值。
