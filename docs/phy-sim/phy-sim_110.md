# 摘要

> 原文：[`phys-sim-book.github.io/lec21.5-summary.html`](https://phys-sim-book.github.io/lec21.5-summary.html)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.4/dist/katex.min.css">

我们通过将势垒能量离散化到边界上的非碰撞点-边缘对，实现了 2D 有限元模拟中的无摩擦自接触，并保证了非相交性。

为了计算势垒能量，我们使用了平方点-边缘距离以避免潜在的数值问题。点-边缘距离是一个分段光滑的函数，其闭式表达式取决于点和边缘的相对位置，整体函数在所有地方都是 C1 连续的。通过应用符号微分到每个表达式，可以方便地获得函数的导数。

对于线搜索滤波，我们不是直接计算容易产生数值问题的碰撞时间（TOI），而是实现了加法 CCD 方法（ACCD）来获得足够大且保守的 TOI 估计。ACCD 是一种迭代方法，它累积 TOI 的下界，同时逐步沿搜索方向推进节点。在运行 ACCD 之前，我们对点和边缘的跨度边界框进行重叠检查，以快速过滤掉非碰撞对。

在后续的讲座中，我们将看到对于 3D 的大规模场景，需要高效的空间索引策略，如空间哈希和边界框层次结构（BVH），以显著降低昂贵的空间搜索成本。

在下一讲中，我们将基于我们刚刚开发的内容实现摩擦自接触。
