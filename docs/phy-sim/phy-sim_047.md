# 摘要

> [原文链接](https://phys-sim-book.github.io/lec8.4-summary.html)



为了减轻在大型时间步长下进行固体模拟时隧穿问题，在整个运动轨迹中强制执行固体与障碍物之间符号距离的非负约束至关重要，而不仅仅是最终解。

虽然直接使用优化路径来近似运动轨迹在理论上并不完美，但它支持设计一种滤波线搜索方案。该方案利用连续碰撞检测（CCD）和投影牛顿法，在实际情况中有效地防止了隧穿现象。

投影牛顿法是一种基于梯度的方法，用于最小化增量势能，它要求势能具有连续的梯度。因此，我们屏障势能中使用的距离函数至少需要是 C1 连续的。对于基于网格的符号距离场（示例 7.1.3），简单的双线性插值被认为是不够的。

此外，处理网格的逐段线性边界上的自接触需要进一步近似以平滑距离函数。对自接触的详细探讨将在未来章节中讨论。在那之前，我们将在下一讲中首先讨论固体-障碍物摩擦。
