# 摘要

> 原文：[`phys-sim-book.github.io/lec4.6-summary.html`](https://phys-sim-book.github.io/lec4.6-summary.html)



我们已经成功演示了实现一个基本的二维质量-弹簧模拟器的实施，该模拟器包含几个关键组件：

+   **网格生成**: 这涉及到节点的创建和连接元素的连接。在实际场景中，模拟器通常从预存的文件中导入网格。

+   **增量势能评估**: 包括计算势能值、其梯度以及对称正定（SPD）投影的 Hessian。

+   **优化时间积分器**: 这包括用于确定搜索方向的线性求解，确保全局收敛的线搜索技术，以及更新节点位置和速度的规则。

+   **模拟器结构**: 包括场景设置、变量初始化和模拟循环的执行。（注意：可视化方面可以与模拟器本身解耦。）

在接下来的章节中，我们将深入探讨边界处理，包括指定的运动和摩擦接触，这些处理通过优化框架中的等式或不等式约束来实现。这一讨论将通过实际案例研究得到丰富，展示每种边界处理在计算模拟中的应用。
