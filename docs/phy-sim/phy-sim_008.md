# 摘要

> 原文：[`phys-sim-book.github.io/lec1.6-summary.html`](https://phys-sim-book.github.io/lec1.6-summary.html)



对固体的模拟涉及预测它们随时间变化的位置和形状。要在计算机上实现这一点，几何形状和时间都必须以离散的形式表示。

几何形状通常通过以特定方式相互连接的样本点来表示：

+   **有限元法 (FEM**) 通过非结构化网格连接样本点。

+   **材料点法 (MPM**) 使用统一的笛卡尔网格来连接样本点。有限元法在提供高精度结果方面表现出色，而材料点法在处理拓扑变化方面具有优势。本书主要关注有限元法。

时间被离散成不同的时刻，使用有限差分法计算物理量的时间导数，符合牛顿第二定律。

由于其无条件不稳定性，通常避免使用前向欧拉法。相反，辛欧拉法是显式的且条件稳定，通常在条件良好的问题中更受欢迎。对于刚性问题，尽管计算强度高且可能存在数值不稳定性，但通常使用无条件稳定的后向欧拉法，因为它需要求解非线性方程组。

在下一讲中，我们将探讨隐式时间积分的优化视角，提供解决这些问题的鲁棒性。
