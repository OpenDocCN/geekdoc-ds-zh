# 分段线性位移场

> [`phys-sim-book.github.io/lec19.1-linear_disp_field.html`](https://phys-sim-book.github.io/lec19.1-linear_disp_field.html)



对于在固体域中的三角形元素，其顶点为 X1，X2 和 X3，我们可以使用空间离散化（参见方程(17.1.1)）来近似该元素中任意点 X 的世界空间坐标：

x^(X)=x(X1)N1(X)+x(X2)N2(X)+x(X3)N3(X),(19.1.1)

该方程表示一个二维插值，扩展了实验示例 17.1.1。在这里，我们假设元素中任意点的世界空间坐标可以通过元素的顶点坐标进行插值。

线性有限元使用方程(19.1.1)中的线性形状函数 Ni，从而在整个域上产生分段线性（每三角形）位移场 u=x^(X)−X。在提供 N 关于 X 的精确表达式之前，我们将引入另一个参数空间以简化推导。

令 β,γ∈[0,1] 且 β+γ≤1，我们可以使用它们来表示元素 X1X2X3 中任意点 X 的材料空间坐标：

X(β,γ)=X1+β(X2−X1)+γ(X3−X1)=(1−β−γ)X1+βX2+γX3.​

在这里，X 是 (β,γ) 的线性函数。使用线性形状函数，x^ 是 X 的线性函数。

回想一下，对于插值，我们必须满足条件 x^(Xi)=x(Xi)。将这些条件综合起来，我们可以获得一个唯一解：

x(β,γ)≈x^(β,γ)=x1+β(x2−x1)+γ(x3−x1)=(1−β−γ)x1+βx2+γx3,​

其中我们用 xi 表示 x(Xi)。这表明：

N1(β,γ)=1−β−γ,N2(β,γ)=β,N3(β,γ)=γ.

有趣的是，有了 X(β,γ)，x(β,γ) 和 N(β,γ) 的表达式，我们不一定需要 x^(X) 和 N(X) 的精确表达式来计算方程(17.2.1)中的每一项。

> ***注 19.1.1（一致性分割）*** 有限元法中的形状函数在每个元素内部满足**一致性分割**：N1(β,γ)+N2(β,γ)+N3(β,γ)=1∀ β,γ∈[0,1] 且 β+γ≤1。有限元的一个优点是，与网格或基于粒子的表示相比，它提供了更精确的边界分辨率。有限元网格的边界节点可以精确地位于连续域的边界上。元素在域内生成，将边界节点连接起来形成离散边界，随着分辨率的提高，该离散边界收敛到连续域的边界。
> 
> 尽管基于粒子的方法也可以在域边界上采样粒子，但它们的球形形状函数会延伸到域外，破坏了单位分割。这会形成一个“软”的物料外层，使得边界力计算不准确。相比之下，有限元方法（FEM）的形状函数仅在各个单元内部非零，且在所有地方都满足单位分割。
