# 空间和时间离散化

> 原文：[`phys-sim-book.github.io/lec26-mpm_disc.html`](https://phys-sim-book.github.io/lec26-mpm_disc.html)



**本讲作者：[张宇](https://changyu.io/)，加州大学洛杉矶分校**

在第一讲中，离散空间和时间，我们介绍了固体对象的多种表示方法。我们不是从经典连续方程开始，而是直接将弹性模拟表述为离散代数问题，之前的讨论主要集中于基于网格的纯拉格朗日方案。

在本讲中，我们介绍了材料点法（MPM）——一种特别适合模拟大变形和拓扑变化的混合拉格朗日-欧拉方案。与像有限元法这样的纯拉格朗日方法不同，后者通过精心构建的网格结构的**元素**跟踪材料变形，MPM 将材料简单地离散成一组**材料粒子**。每个粒子存储基本物理信息，如质量、动量、变形梯度和应力。然后，MPM 利用一个临时的、规则的笛卡尔背景网格来解决动量方程，并在粒子与网格节点之间进行信息传递。

具体来说，在每个时间步中，首先将材料粒子的信息传递到欧拉网格上，在那里高效地求解控制方程。然后将更新的网格解传递回粒子，更新粒子状态，如位置和变形梯度。在此传递之后，网格被重置，留下粒子作为材料历史的唯一载体。这种混合策略自然地适应了显著的变形和复杂的接触场景，有效地克服了传统基于网格的纯拉格朗日方法固有的典型网格畸变挑战。
