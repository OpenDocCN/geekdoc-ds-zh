# 概述

> 原文：[`phys-sim-book.github.io/preface.html`](https://phys-sim-book.github.io/preface.html)



这本免费在线书籍标志着我们致力于使基于物理的模拟的理论和算法对每个人可访问。

## 贡献

如果您有兴趣为编辑和改进此书做出贡献，请通过 [mdbook-src](https://github.com/phys-sim-book/mdbook-src) 仓库的 Github 拉取请求进行，*(不是 HTML 仓库)*，或者直接联系 [李敏辰](https://www.cs.cmu.edu/~minchenl/) 和 [姜晨帆](https://www.math.ucla.edu/~cffjiang/)。

根据您贡献的性质，您将在书籍未来的构建中列为合著者或社区贡献者。
