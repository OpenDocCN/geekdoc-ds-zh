# 超越弹性：塑性和粘性

> 原文：[`phys-sim-book.github.io/lec27-mpm_plasticity.html`](https://phys-sim-book.github.io/lec27-mpm_plasticity.html)



**本讲座作者：[张宇](https://changyu.io/), 加州大学洛杉矶分校**

在基于物理的模拟中，**连续性假设**使我们能够用丰富多样的行为来模拟广泛的材料。虽然**超弹性**为那些在变形后能恢复到其原始形状的材料提供了一个方便且优雅的模型，例如橡胶或软组织，但它只代表了在现实世界中观察到的物理现象的一小部分。

许多自然和日常材料，包括**沙子**、**雪**、**泥**、**金属**、**泡沫**和**断裂固体**，表现出不可逆的变形、耗散或类似流动的行为，这些行为不能仅通过弹性模型来捕捉。这些材料需要包含**塑性**（永久变形）和**粘性**（速率依赖性应力响应）的本构模型。

**弹性**和**塑性**之间的核心区别在于材料存储和耗散能量的方式。弹性变形是可逆的和能量守恒的；塑性变形是不可逆的，受**屈服准则**控制，并涉及通过材料结构的内部重新配置来**耗散能量**。

MPM 特别适合模拟塑性和粘性材料，因为它自然支持大变形、历史依赖状态更新和局部非弹性流动，而不会遭受传统基于网格的方法中常见的网格纠缠、单元倒置或重新网格化相关的伪影。

在本讲座的剩余部分，我们将介绍如何将塑性和粘性纳入 MPM 模拟。我们首先介绍塑性流的离散化和不可逆变形的表示。然后描述如何实施材料特定的屈服条件并执行回映射以限制变形在可接受范围内。
