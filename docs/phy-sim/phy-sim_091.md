# 非穿透的正常接触

> [`phys-sim-book.github.io/lec18.2-normal_contact.html`](https://phys-sim-book.github.io/lec18.2-normal_contact.html)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.4/dist/katex.min.css">

为了防止在模拟过程中发生自穿透，强制一个条件确保变形映射ϕ(⋅,t):Ω0→Ωt 对于任何 t≥0 都是双射的至关重要。这种双射由作用于接触表面区域对的边界力维持，称为ΓC。我们可以将这些力视为另一组诺伊曼边界条件，仅在必要时对ΓC 施加额外的力以防止穿透。因此，我们可以将弱形式中的边界积分项扩展如下：

∫∂Ω0Qi(X,t)Ti(X,t)ds(X)=∫ΓDQi(X,t)TD∣i(X,t)ds(X)+∫ΓNQi(X,t)TN∣i(X,t)ds(X)+∫ΓCQi(X,t)TC∣i(X,t)ds(X), (18.2.1)

其中 TN(X,t)是在问题设置中指定的原始诺伊曼边界力，而 TC(X,t)是由双射约束产生的法向接触力。

与狄利克雷边界条件类似，TC(X,t)只能在解决问题后确定。然而，强制非穿透比规定位移更复杂。幸运的是，我们可以使用 IPC 中 TC(X,t)的近似本构模型来表示接触力作为 x 的函数，通过简单地包括这个额外的保守力来确保非穿透。

> ***备注 18.2.1（重叠边界）。*** 注意，在这里ΓC 可以与ΓD 和ΓN 都重叠。对于一个自由（诺伊曼）边界与狄利克雷边界接触，当强制狄利克雷边界条件时，狄利克雷部分的 TC(X,t)也将被忽略。然而，如果两个狄利克雷边界相互穿透，问题将因双射约束而无解。
