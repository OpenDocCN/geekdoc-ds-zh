在连续设置中，力被分为体积力（也称为外力，例如重力）和表面力（或内力，通常是基于应力的，如由弹性引起的那些）。我们通过牵引场定义基于应力的力，其存在是假设的。牵引，或单位面积上的力，由场 T(⋅,N,t):Ω0→Rd 表示，并由以下方程定义：力 S(Bϵ0)=∫∂Bϵ0T(X,N(X))ds(X)，其中 N 表示在材料空间中的指向外的法线方向。在这里，力 S(Bϵ0)表示从 ∂Bϵ0 外的材料对 Bϵ0 内的材料通过它们的界面施加的净力。函数 T(X,N,t) 量化了 N+ 侧的材料在 X 点对 N− 侧材料的单位面积（d=3）或长度（d=2）上的力。

可以证明这表明存在一个应力场（第一 Piola-Kirchoff 应力）P(⋅,t):Ω0→Rd×d，其中：T(X,N,t)=P(X,t)N。

然后，通过在 Bϵ0 上应用牛顿第二定律，我们可以将动量守恒表示为：=∫Bϵ0R(X,0)∂t/∂V(X,t)dX∫∂Bϵ0P(X,t)N(X)ds(X)+∫Bϵ0R(X,0)Aext(X,t)dX，对于所有 Bϵ0⊂Ω0 和 t≥0。

应用散度定理，我们可以将方程 (16.2.1) 中的边界积分转换为体积积分，并得到：=∫Bϵ0R(X,0)∂t/∂V(X,t)dX∫Bϵ0∇X⋅P(X,t)dX+∫Bϵ0R(X,0)Aext(X,t)dX，对于所有 Bϵ0⊂Ω0 和 t≥0。

> ****定义 16.2.1（向量散度定理）**** 对于在封闭域 Ω 上定义的向量值函数 f(x):Ω→Rd，令 n(x) 为该域边界的指向外的法线，以下等式成立：∫∂Ωf⋅nds(x)=∫Ω∇⋅fdx。这个定理使我们能够方便地在边界和体积积分之间转换。

在这里，散度算子 ∇⋅ 对 P 的每一行向量独立作用，并产生一个列向量：(∇X⋅P)i=∑jPij,j。由于方程 (16.2.2) 也适用于任意的 Bϵ0，通过移除积分，我们得到力平衡方程的强形式：R(X,0)∂t/∂V(X,t)=∇X⋅P(X,t)+R(X,0)Aext(X,t)，对于所有 X∈Ω0 和 t≥0。

> ***备注 16.2.1（固体模拟中的动量守恒）*** 动量守恒是我们用来模拟固体的主要控制方程。如前所述，加速度，表示为 ∂t/∂V(X,t)，和内力，表示为 ∇X⋅P(X,t)，都可以使用世界空间坐标 x 来描述。在建立所有其他相关量之后，我们逐步求解 x 以获得动态运动的每一步。
