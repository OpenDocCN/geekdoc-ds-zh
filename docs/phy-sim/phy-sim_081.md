# 质量守恒

> 原文：[`phys-sim-book.github.io/lec16.1-mass_conserv.html`](https://phys-sim-book.github.io/lec16.1-mass_conserv.html)



我们可以将质量密度 R(X,t) 视为在 Ω0 上自然定义的，即 R(X,t) = ϵ→+0lim 体积(Bϵt) / 质量(Bϵt) = ϵ→+0lim ∫Bϵt dx / 质量(Bϵt) ，(16.1.1) 其中 Bϵt 是 Bϵ0 的世界空间对应物（围绕 Ω0 中的任意 X∈Ω0 的半径为 ϵ 的球体）。这可以说是自然的定义，因为质量(Bϵt) 应该是一个可测量的量。质量守恒可以表示为 质量(Bϵt) = 质量(Bϵ0) ，∀ Bϵ0 ⊂ Ω0 且 t≥0。(16.1.2)

现在，通过变量替换，我们得到 ∫Bϵt dx = ∫Bϵ0 J(X,t)dX，因此方程 (16.1.1) 变为 R(X,t) = ϵ→+0lim ∫Bϵ0 J(X,t)dX / mass(Bϵt) ，(16.1.3) 因此 R(X,0) = ϵ→+0lim ∫Bϵ0 dX / mass(Bϵ0) ，(16.1.4) 因为 J(X,0) = 1。然后结合方程 (16.1.2)，(16.1.3) 和 (16.1.4)，我们可以将质量守恒表示为 ∫Bϵ0 R(X,t)J(X,t)dX = ∫Bϵ0 R(X,0)dX ，∀ Bϵ0 ⊂ Ω0 且 t≥0。这仅仅意味着 Bϵt 中的质量（通过质量密度的积分表示）不应随时间变化。这个集合与时间 t 的材料的一个子集相关联，随着它在流动中的演化，材料将占据更多或更少的空间，但集合中始终会有相同数量（质量）的材料。由于 Bϵ0 是任意的，因此对于所有 X∈Ω0 且 t≥0，必须有 R(X,t)J(X,t) = R(X,0)。

> ***备注 16.1.1 (拉格朗日和欧拉视角).*** 在基于 Ω0 直接离散化和追踪材料的仿真方法中，质量守恒是固有的。例如，在我们的有限元方法 (FEM) 仿真器中，Ω0 被分割成三角形，在整个仿真过程中，每个三角形的质量保持恒定，不受变形的影响。这种方法被称为**拉格朗日方法**。相比之下，**欧拉方法**基于 Ωt 离散化和演化物理量，通常需要特别处理质量守恒问题。
