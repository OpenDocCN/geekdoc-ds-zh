# 摘要

> 原文：[`phys-sim-book.github.io/lec25.10-summary.html`](https://phys-sim-book.github.io/lec25.10-summary.html)



在本讲座中，我们讨论了模态缩减技术，这些技术通过将高维系统投影到精心选择的低维子空间，显著加速了基于物理的模拟。

线性模态分析使用物体的自然振动模式作为基向量，创建独立的 1D 振荡器，求解效率极高。然而，由于线性假设，这种方法在大变形时会产生严重的伪影（长度拉伸、体积变化）。非线性模型缩减将这些概念扩展到处理大变形，通过投影完整的非线性运动方程。关键挑战是高效地评估缩减力，而不需要计算全空间量，这通过特定材料的多项式拟合或一般情况下的求积方法来解决。

任何模态缩减的质量都严重依赖于基函数的选择。存在两种主要方法：使用主成分分析对模拟快照进行数据驱动的方法，以及基于物理的模态导数，它捕捉线性模式之间的非线性耦合，而不需要昂贵的预模拟。
