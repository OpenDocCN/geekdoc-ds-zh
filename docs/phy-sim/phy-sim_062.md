# 连续体运动

> 原文：[`phys-sim-book.github.io/lec12.1-continuum_motion.html`](https://phys-sim-book.github.io/lec12.1-continuum_motion.html)



运动学是研究连续体材料内运动的研究，主要关注在不同坐标系中，局部或全局发生的形状或变形变化。目标是定性定量地描述运动，这对于推导动力学和机械响应的控制方程至关重要。值得注意的是，运动学通常可以在不引入诸如力、应力或甚至质量等概念的情况下进行描述。

在连续体力学中，变形通常通过三个关键组成部分来表示：

+   **材料（或未变形）空间 X**：这代表材料中任何点的初始位置。

+   **世界（或变形）空间 x**：这表示任何点的当前位置。

+   **变形映射 ϕ(X,t)**：这个函数将材料空间中的点映射到世界空间中，显示了材料点随时间位置的变化。

在初始时间 t=0，材料空间 X 和世界空间 x 重合，这意味着每个点都从其未变形的位置开始。

> ****定义 12.1.1（变形/流动映射）。**** 连续体力学中材料的运动由映射 ϕ(⋅,t):Ω0→Ωt 确定，其中 Ω0,Ωt⊂Rd，d=2 或 3 代表模拟问题（或域）的维度。这个映射，通常被称为流动映射或变形映射，对于理解材料点随时间移动至关重要。
> 
> +   **材料点 X**：集合 Ω0 中的点被称为材料点，并指定为 X。
> +   
> +   **当前位置 x**：Ωt 中的点代表时间 t 时材料点的位置，被称为 x。变形映射 ϕ 描述了每个材料点 X∈Ω0 在整个时间内的轨迹，表示为：x=x(X,t)=ϕ(X,t)。
> +   
> ****示例 12.1.1.**** 如果我们的物体以恒定速度 v 沿方向 n 运动，那么我们有 x=X+tvn。(12.1.1) 如果物体在时间 t（与时间 0 相比）之后经历某种刚性运动，我们将有 x=RX+b,(12.1.2) 其中 R 是一个旋转矩阵，b 是某种平移。R 和 b 很可能随时间 t 和初始位置 X 的变化而变化，具体取决于实际的运动。

映射 ϕ 可以用来量化相关的基于连续体的物理学。例如，给定材料点 X 在时间 t 的速度是 V(X,t)=∂t∂ϕ(X,t),(12.1.3) 而加速度是 A(X,t)=∂t2∂2ϕ(X,t)=∂t∂V(X,t).(12.1.4) 这意味着 V(⋅,t):Ω0→Rd 和 A(⋅,t):Ω0→Rd。

> ***备注 12.1.1.*** 在上述内容中，速度 V 和加速度 A 是从拉格朗日视角定义的。这意味着速度和加速度都是材料配置 X 和时间 t 的函数，专注于材料中的特定粒子。从物理学的角度来看，这表明这些测量涉及具有自身质量和从模拟开始就占据了一些体积的粒子。拉格朗日视角对于跟踪单个粒子随时间的动力学变化特别有价值，提供了深入了解粒子如何在各种条件下在材料中移动、加速和相互作用的详细信息。
