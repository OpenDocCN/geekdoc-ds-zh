# 摘要

> 原文：[`phys-sim-book.github.io/lec2.5-summary.html`](https://phys-sim-book.github.io/lec2.5-summary.html)



我们到目前为止讨论的目标是设计一个可靠的解决方案来解决无条件的稳定隐式时间积分问题。我们旨在解决由截断误差引起的非收敛解的问题。我们通过将时间积分问题重新表述为最小化问题来解决这个问题。这种表述不仅允许我们应用成熟的优化技术，而且还为不同物理现象提供了一个一致建模框架。

在此框架内，用于模拟各种现象的技术有以下简要总结：

+   对于像重力和弹性这样的保守力，我们使用了势能。这些势能被整合到目标函数中，以准确表示涉及的力。

+   边界条件，它指定了系统的约束，是通过简单的线性等式约束来建模的。这有助于我们在进行模拟时将系统限制在可行状态。

+   为了防止在模拟过程中固体对象之间的相互穿透，我们使用了不等式约束来建模接触和摩擦。这些约束确保了物体在相互接触时保持其物理完整性，并表现出预期的行为。

这里需要注意的一个重要方面是，我们可以利用边界条件的独特结构以无约束的方式强制执行等式约束。这将导致计算复杂性的显著降低。

此外，我们引入了增量势能接触（IPC）方法。IPC 方法将接触和摩擦建模为具有可控精度的平滑势能。这确保了对固体对象的稳健和准确模拟，避免了相互穿透。

在接下来的讲座中，我们将深入探讨用于解决无约束优化问题的投影牛顿法。这种方法具有全局收敛的优势，这意味着只要初始配置是可行的，该方法就能保证收敛，无论初始配置如何。这一特性对于复杂模拟来说非常理想，它有助于使该方法更加稳健和可靠。
