# 摘要

在本节中，我们探讨了求解在基于优化的模拟中出现的形如 Ax=b 的大型稀疏线性系统的基本方法，特别是在计算牛顿型方法中的搜索方向时。

我们确立了上下文：在投影牛顿方法的每次迭代中，我们求解 Pp=−∇E(x)，其中 P 通常是典型的对称正定（SPD）代理矩阵。这简化为求解对优化时间积分器至关重要的线性系统。

直接求解器通过矩阵分解提供精确解。对于 SPD 系统，Cholesky 分解 A=LLT 将问题简化为两个三角求解。虽然稳健且准确，但由于计算限制，直接方法对于非常大的系统来说变得不切实际。

迭代方法为大型稀疏系统提供了替代方案。基本方法包括雅可比（自然可并行化）和高斯-赛德尔（收敛速度更快但顺序执行）。当它们的迭代矩阵的谱半径小于 1 时，两者都会收敛。共轭梯度（CG）方法是 SPD 系统的一种更复杂的方法。CG 构建 A 共轭搜索方向，确保进展不会被撤销，通过正交关系实现显著的效率。在精确算术中，CG 最多在 n 步内收敛，而在实践中，它通常在预条件设置下收敛得更快。

直接方法和迭代方法的选择取决于问题规模、稀疏结构和精度要求。对于需要高精度的中等规模问题，直接方法表现优异，而迭代方法，尤其是带有预条件的 CG 方法，对于大规模模拟至关重要。
