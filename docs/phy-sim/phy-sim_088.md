在本讲座中，我们在空间和时间上对动量守恒的弱形式进行了离散化，得到了在第一讲中引入的向后 Euler 时间积分的方程组。

**空间离散化：**

对于空间离散化，在域内采样有限数量的点，并使用它们的位移作为模拟的自由度（DOF）。与每个自由度相关的插值函数，可以近似地确定域内任何点的位移，将弱形式的解限制在由相互正交的插值函数形成的 d n 维函数空间中，其中 n 代表采样点的数量。这样，测试函数 Q 可以方便地分配，以生成 nd 个方程来求解 nd 个未知数。

**时间离散化：**

时间离散化将加速度 A 通过特定的时间积分规则与自由度 x 连接起来。通过应用质量集中和假设零牵引边界条件，我们最终可以推导出离散形式。插值函数的积分将在下一章中介绍。

在下一讲中，我们将讨论连续设置中的边界条件和摩擦接触。
