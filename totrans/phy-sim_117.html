Similar to 2D triangle elements, we use β,γ,τ∈[0,1] with β+γ+τ≤1 to express the material space coordinates of an arbitrary point X in the tetrahedral element defined by vertices X1​,X2​,X3​, and X4​ as follows: X(β,γ,τ)​=X1​+β(X2​−X1​)+γ(X3​−X1​)+τ(X4​−X1​)=(1−β−γ−τ)X1​+βX2​+γX3​+τX4​.​ Here, X is a linear function of (β,γ,τ). Using linear shape functions, the approximate world-space coordinate x^ is also a linear function of (β,γ,τ): x(β,γ,τ)≈x^(β,γ,τ)=(1−β−γ−τ)x1​+βx2​+γx3​+τx4​,​ where x(Xi​) is denoted as xi​. This implies that the shape functions are: N1​(β,γ,τ)N2​(β,γ,τ)N3​(β,γ,τ)N4​(β,γ,τ)​=1−β−γ−τ,=β,=γ,=τ.​
